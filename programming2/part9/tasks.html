Uppgifterna nedan bygger på avsnittet om Del 9 - Overloading, Overriding, Polymorfism i Python.

Uppgift 1.2: Simulera en överladdad metod med **kwargs ​

Skapa en klass Rektangel med en metod beräkna_area som kan hantera två typer av argument: antingen längd och bredd eller en sida (för att skapa en kvadrat). Använd **kwargs för att göra metoden flexibel.

Instruktioner:

  1. Använd **kwargs för att ta emot antingen längd och bredd, eller bara en sida.
  2. Om längd och bredd finns i kwargs, returnera arean som längd * bredd.
  3. Om endast sida finns, returnera arean som sida * sida.
python
class Rektangel:
    def beräkna_area(self, **kwargs):
        # Skriv din kod här
        pass

# Testa metoden med både rektangel och kvadrat
rektangel = Rektangel()
print(rektangel.beräkna_area(längd=5, bredd=10))  # Förväntat resultat: 50
print(rektangel.beräkna_area(sida=7))             # Förväntat resultat: 49
