Installera Pyhton-tolk i VS Code steg för steg ​

Vi hade laddat ner Python från dess hemsida. Nästa steg är att koppla rätt Python-tolk (Interpreter) till VS Code.

Installation ​

För att installera Python-tolk(interpreter):

  1. Öppna VS Code.

  2. Menyfliken View
    Klicka på VS Code menyfliken View och välj Command Palette eller tryck ner på tangenterna Ctrl+Shift+P:


  3. Command Palettes sökfält
    Ange i Command Palettes sökfält följande: Python: Select Interpreter


  4. Välj rätt Pythons tolk
    Välj rätt Pythons tolk (interpreter), den du laddade ner från Pythons hemsida:
