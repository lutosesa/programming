Introduktion ​

Python är ett av de mest populära och användarvänliga programmeringsspråken idag, särskilt inom utbildning. För gymnasiestudenter som lär sig programmera, är Python ett utmärkt språk att börja med på grund av sin enkelhet och tydlighet. Den här handledningen är en repetition av grundläggande Python-ämnen och begrepp från "Programmering 1" och den sträcker sig fram till ämnet funktioner. Vi kommer att gå igenom definitioner, förklaringar och praktiska exempel som hjälper dig att fräscha upp och befästa dina kunskaper.

Målet med detta avsnitt: ​

  1. Variabler och Datatyper

1. Variabler och Datatyper ​

Vad är en variabel? ​

En variabel är en behållare för att lagra data. I Python behöver du inte specificera datatypen när du skapar en variabel; Python tilldelar automatiskt datatypen baserat på värdet som du ger variabeln. En variabel är egentligen en plats i datorns minne där du kan lagra data. Variabler gör det möjligt att referera till data genom ett namn istället för att använda minnesadresser.

Datatyper ​

  • Heltal (int): Ett helt nummer. Exempel: 18
  • Flyttal (float): Ett nummer med decimaler. Exempel: 3.14159.
  • Strängar (str): Text, omsluten av enkla eller dubbla citattecken. Exempel: "Hej", 'Världen'
  • Boolean (bool): Ett värde som kan vara True eller False.

Exempel ​

Ex.

python
namn = "Kalle"
ålder = 18
poäng = 35.5
är_elev = True

print(f"{namn} är {ålder} år.")
print(f"\nÄr han elev på Rudbeck? \n{'Ja' if är_elev else 'Nej'}")
print(f"\n{namn} fick {poäng} poäng på matte-provet!")

2. Operatorer ​

Operatorer används för att utföra operationer på variabler och värden.

Matematiska operatorer: ​

  • Addition: +
  • Subtraktion: -
  • Multiplikation: *
  • Division: /
  • Exponentiering: ** (t.ex. 2**3 ger 8)
  • Modulus (resten av en division): %

Exempel ​

Ex.

python
a = 10
b = 3

summa = a + b     # 13
skillnad = a - b  # 7
produkt = a * b   # 30
kvot = a / b      # 3.3333
rest = a % b      # 1
 
print(f"Summa = {summa}")
print(f"Skillnad = {skillnad}")
print(f"Produkt = {produkt}")
print(f"Kvot = {kvot:.2f}") 
print(f"Rest = {rest}")

Jämförelseoperatorer: ​

Dessa operatorer används för att jämföra två värden. Resultatet av en jämförelse blir sant (True) eller falskt (False).

  • Lika med: ==
  • Inte lika med: !=
  • Större än: >
  • Mindre än: <
  • Större än eller lika med: >=
  • Mindre än eller lika med: <=
python

print(x != y)  # True
print(x > y)   # True

print(x <= y)  # False

4. Loopar ​

Loopar används för att upprepa en kodsekvens flera gånger.

While-loop: ​

Körs så länge ett villkor är sant.

Exempel 1 ​

Exempel med användarinmatning:

python
password = ""
while password != "python123":
    password = input("<Ange lösenordet> ")
print("Tillträde beviljad!")
1
2
3
4

Förklaring:

  • Variabeln password sätts initialt till en tom sträng.
  • Loopen fortsätter att be användaren om att skriva in lösenordet tills det korrekta lösenordet "python123" skrivs in.
  • När rätt lösenord skrivs in, avslutas loopen och "Tillträde beviljad!" skrivs ut.

Exempel 2 ​

Exempel på att undvika oaändlig loop

python
while True:
    user_input = input("Skriv 'stopp' för att avsluta: ")
    if user_input.lower() == 'stopp':
        break
    print(f"Du skrev: {user_input}")
print("Programmet är avslutat.")
1
2
3
4
5
6

Förklaring:

  • Loopen körs oändligt eftersom villkoret alltid är True.
  • Användaren ombeds skriva något.
  • Om användaren skriver "stopp", körs break och loopen avslutas.
  • Om användaren skriver något annat, fortsätter loopen.

Exempel 3 ​

Exempel på att räkna ned från 5 till 0.

python
# Startvärde för nedräkningen
n = 5

# While-loop för att räkna ned från 5 till 0
while n >= 0:
    print(n)
    n -= 1  # Minska värdet av n med 1 vid varje iteration
1

Förklaring:

  • Variabeln n sätts till 5, som är startvärdet för nedräkningen.
  • En while-loop används för att utföra nedräkningen. Loopen fortsätter så länge som värdet på n är större än eller med 0.
  • Inne i loopen skrivs det aktuella värdet av n ut.
  • Efter att talet skrivits ut minskas n med 1 (n -= 1).
  • När n blir mindre än 0 avbryts loopen, och nedräkningen är klar.

6. Funktioner ​

Funktioner är block av kod som utför en specifik uppgift och kan återanvändas flera gånger. Funktionen kan ta emot indata, bearbeta den och sedan returnera ett resultat.

Skapa en funktion ​

python
def hej():
    print("Hej!")
1
2

Anropa en funktion ​

python
def hej():
    print("Hej!")

hej()  # Skriver ut "Hej!"
