Projekt 4 : Potentiometer ​

Inledning ​

Genom att ansluta en potentiometer till det analoga pin A0 i kretskortet Arduino Uno, visas hur snabb lysdioden tänds och släcks beroende på värdet som lagras och tilldelas med funktionen analogRead(PIN_POTENTIOMETER) i variabeln analogValue. Värdet på variabeln är från 0 till 1023. Lysdioden tänds och släcks fortare eller långsammare om man vrider på potentiometern.

Syfte med projektet ​

Vi kommer att lära oss om:

  • hur globala konstanta variabler deklareras och tilldelas
  • hur vanliga variabler deklareras, tilldelas och används
  • hur används de huvudfunktionerna och andra funktioner som ingår i programmet
  • hur man kopplar in en lysdiod, en resistor, och en potentiometer
  • hur man kan styra dessa elektroniska komponenter genom att koda fram en algoritm på utvecklingsmiljön Arduino IDE.

Vad är en potentiometer? ​

En potentiometer är en enhet som används för att mäta och kontrollera spänning. Genom att vrida på potentiometern kan du ändra på spänningen som Arduino kan läsa via en analog ingång (t.ex. A0). Den används ofta för att justera värden som volym, ljusstyrka eller hastighet i projekt. Den har tre ben:

  • De yttre benen av potentiometern är anslutna till spänning och jord.
  • Mitten benet, ger skillnaden i spänning när du vrider på ratten.

Elektroniska delar som behövs ​

  • (1) Arduino Uno
  • (1) USB A-to-B kabel
  • (1) Kopplingsplatta
  • (1) Lysdiod (LED)
  • (1) 220 Ω Resistor
  • (1) 1KΩ Resistor
  • (1) Potentiometer
  • sladdar

Övning 1 ​

Vi ska ansluta en potrntiometer en röd Lysdiod och en resistor på den vita kopplingsplattan till kretskortet Arduino Uno. Lysdioden ska tändas och släckas fortare eller långsammare om man vrider på potentiometern.

Kopplings ritning ​

Programkod ​

cpp
/****************************************************
* Projekt 4 - övning 1 - Potentiometer
* 
* Programmet styr en blink-hastighet i en 
* lysdiod med en potentiometer.
*****************************************************/


// skapar konstanta variabler för potentiometer och lysdiod.
const int PIN_POTENTIOMETER = A0;   // pin-värde för potentiometer.
const int PIN_LED = 13;             // pin-värde för lysdioden.


// skapar variabeln för att lagra värde från potentiometerns pin A0
int analogValue  = 0;    

void setup()
{
  // Lysdioden som UTSIGNAL
  pinMode(PIN_LED, OUTPUT);

  // för att visa info på monitorn.
  Serial.begin(9600);
}

void loop() 
{
  // lagrar ett värde som kommer från potentiometern (0-1023)
  analogValue = analogRead(PIN_POTENTIOMETER);

  // visar data på monitorn
  Serial.print("analogValue = ");
  Serial.print(analogValue);
  Serial.println(" millisekunder");

  digitalWrite(PIN_LED, HIGH);
  delay(analogValue);
  digitalWrite(PIN_LED, LOW);
  delay(analogValue); 
}

Övning 2 ​

Nu ska du ansluta en lysdiod till kretsen. Det vill säga, du ska en potentiometer och två lysdioder. Den ena lysdioden ska tändas och släckas fem gånger snabbare än den andra lysdioden. Du ska visa värdena på hur snabb dessa dioder blinkar på Arduino IDEs inbyggda monitor.

Kopplings ritning ​

Programkod ​

cpp
/************************************************************
* Projekt 4 - övning 2 - Potentiometer
*
* Programmet kan styra med potentiometer en blink-hastighet i 
* ena lysdioden och 5 gånger snabbare i den andra lysdioden. 
* 
* Analog-pin A0 avger en analog signal mellan 0 och 1023 
* via potentiometern. Detta värde blir blink-fördröjningen 
* i lysdioden. 
**************************************************************/

// skapar konstanta variabler för potentiometer och lysdiod.
const int PIN_POTENTIOMETER = A0;   // pin-värde för potentiometer.
const int PIN_LED_13 = 13;          // pin-värde för lysdioden.
const int PIN_LED_5 = 5;

void setup() 
{
  pinMode(PIN_LED_13, OUTPUT);
  pinMode(PIN_LED_5, OUTPUT);
  
  // för att visa info på monitorn.
  Serial.begin(9600);
}

void loop() 
{
  // lagrar värde från potentiometer
  int analogValue = analogRead(PIN_POTENTIOMETER);
 
  // visar data på monitorn
  Serial.print("analogValue-13 = ");
  Serial.print(analogValue);
  Serial.println(" millisekunder");
  
  Serial.print("analogValue-5 = ");
  Serial.print(analogValue/5);
  Serial.println(" millisekunder");
  
  digitalWrite(PIN_LED_13, HIGH);
  delay(analogValue);
  digitalWrite(PIN_LED_13, LOW);
  delay(analogValue); 
  
  digitalWrite(PIN_LED_5, HIGH);
  delay(analogValue/5);
  digitalWrite(PIN_LED_5, LOW);
  delay(analogValue/5);  
}
