Projekt 3 : Tryckknappsbrytare ​

Inledning ​

Du ska ansluta en tryckknappsbrytare till en lysdiod och en resistor på den vita kopplingsplattan till kretskortet Arduino Uno.

Lysdioden ska tändas när tryckknappsbrytare trycks ner och släckas när tryckknappsbrytare släpps. Sedan kommer du att koppla in flera lysdioder och tryckknappar.

Du kommer att använda if och else för att kontrollera om tryckknappen är intryck.

Syfte med projektet ​

Du kommer att lära dig om:

  • hur if och else fungerar,
  • hur globala konstanta variabler deklareras och tilldelas,
  • hur vanliga variabler deklareras, tilldelas och används,
  • hur man kopplar in en lysdiod, en resistor, och en tryckknappsbrytare,
  • hur används de huvudfunktionerna och andra funktioner som ingår i programmet,
  • hur man kan styra dessa elektroniska komponenter genom att koda fram en algoritm på utvecklingsmiljön i Tinkercad.

Vad är en tryckknappsbrytare? ​

En tryckknappsbrytare är en enkel mekanisk brytare som används för att registrera ett på/av-läge (digital signal). När knappen trycks ned, sluter den en krets och skickar en signal till en digital ingångspinne på Arduino (t.ex. pin D2), vilket gör att Arduino kan upptäcka att knappen har tryckts. Den används ofta i projekt för att ge användarinteraktion, som att starta eller stoppa funktioner.

Vad är If och else ? ​

I Arduino används if och else för att skapa villkorssatser, som låter programmet fatta beslut baserat på om vissa villkor är sanna eller falska.

  • if: Utför kod endast om det angivna villkoret är sant.
  • else: Utför alternativ kod om det tidigare villkoret i if var falskt.

Exempel

cpp
// läser om tryckknappen är intryck eller inte:
buttonState = digitalRead(PIN_BUTTON);

// checkar om tryckknappen är intryck (HIGH=1)
if (buttonState == HIGH) {
    digitalWrite(PIN_LED, HIGH);  // lysdioden tänds:
}
else {
    digitalWrite(PIN_LED, LOW);  // lysdioden släcks:
}

Övning 1 ​

Du ska ansluta en tryckknappsbrytare en röd Lysdiod och en resistor på den vita kopplingsplattan till kretskortet Arduino Uno. Lysdioden ska tändas när tryckknappsbrytare trycks ner och släckas när tryckknappsbrytare släpps.

Kopplings ritning ​

Programkod ​

cpp
/*********************************************
* Projekt 3 - övning 1 - Tryckknappbrytaren
* 
* Slår på och av en lysdiod (LED) ansluten 
* till digital pin 10, när du trycker på en 
* tryckknappbrytare fäst på pin 2.
***********************************************/


// skapar konstanta variabler för knappens pinne och lysdiodens pinne
const int PIN_BUTTON = 2;  
const int PIN_LED    = 10; 

// skapar variabeln för att läsa knappens läge/tillstånd
int buttonState = 0;    

void setup()
 {
  // sätts lysdiodens pinne som UTSIGNAL
  pinMode(PIN_LED, OUTPUT);
  
  // sätts knappens pinne som INSIGNAL
  pinMode(PIN_BUTTON, INPUT);
}

void loop() 
{
  // läser om tryckknappen är intryck eller inte:
  buttonState = digitalRead(PIN_BUTTON);

  // checkar om tryckknappen är intryck (HIGH=1)
  if (buttonState == HIGH) 
  {
    digitalWrite(PIN_LED, HIGH);  // lysdioden tänds
  }
  else 
  {
    digitalWrite(PIN_LED, LOW);  // lysdioden släcks
  }
}
