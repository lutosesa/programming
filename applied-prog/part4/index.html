Laboration 1 : Blink med Arduino i Tinkercad ​

För att genomföra en laboration med det klassiska "Blink"-exemplet i Tinkercad, som är en webbaserad plattform för att simulera Arduino-projekt, följer här en steg-för-steg-guide. Denna laboration är utformad för dig, gymnasieelev, och ger en grundläggande förståelse för hur man använder Tinkercad för att simulera Arduino-kretsar.

Mål ​

Att simulera och förstå hur ett Arduino-program kan styra en LED (lysdiod) att blinka med en given tidsintervall.

Material ​

  • En dator med internetuppkoppling.
  • Ett konto på Tinkercad (gratis registrering).
  • Grundläggande kunskaper i C++ och Arduino.

Steg 1: Skapa ett Tinkercad-konto ​

  • Gå till Tinkercad.
  • Registrera dig för ett gratis konto eller logga in om du redan har ett.

Steg 2: Starta ett nytt Arduino-projekt ​

  • Klicka på "Circuits" i Tinkercad-dashboarden.
  • Klicka på "Create new Circuit".
  • Dra en Arduino Uno R3 från komponentbiblioteket till arbetsytan.
  • Dra en LED och ett 330 ohm motstånd från komponentbiblioteket till arbetsytan.

Steg 3: Koppla upp kretsen ​

  1. Anslut LED:

    • Placera LED på arbetsytan. Notera att LED har två ben: det längre benet är anoden (positivt) och det kortare är katoden (negativt).
    • Anslut det längre benet (anoden) till digital pin 13 på Arduino.
    • Anslut det kortare benet (katoden) till ena änden av 330 ohm motståndet.
    • Anslut den andra änden av motståndet till GND (jord) på Arduino.

  2. Dubbelkolla att alla anslutningar är korrekta.

Steg 4: Programmera Arduino ​

  1. Klicka på "Code" längst upp till höger på arbetsytan för att öppna kodredigeraren.

  2. Se till att Text är vald i kodredigeraren.

  3. Skriv (kopiera) in följande kod nedan för att få LED att blinka:

cpp
/****************************************************
* Laboration 1- Blink en lysdiod - övn 1
* 
* Programmet/sketchen går ut på att en röd lysdiod 
* tänds och släcks en sekund inom en funktion som 
* kör för alltid i en loop.
*****************************************************/

// Global variabel.
// Tilldelning av global variabel med värdet 13.
int PIN_LED_RED = 13; 

// vanlig variabel
int timer = 1000;

void setup()
{
  // Funktionen pinMode() anropas för att ställa in stift 13 som UTSIGNAL.
  pinMode(PIN_LED_RED, OUTPUT);
}

void loop()
{  
  digitalWrite(PIN_LED_RED, HIGH); 	//Lysdioden tänds 
  delay(timer);						//väntar en sekund. 
  digitalWrite(PIN_LED_RED, LOW);	//Lysdioden släcks.
  delay(timer);						//väntar en sekund.
}

Steg 5: Simulera projektet ​

  1. Klicka på "Start Simulation" för att köra simuleringen.
  2. Om allt är rätt kopplat och programmerat, ska LED nu blinka med 1 sekunds intervall.

Steg 6: Utforska och Experimentera ​

  • Ändra delay(1000); till ett annat värde, som delay(500);, för att se hur blinkhastigheten förändras.

Steg 7: Lägg till flera LED-lampor ​

  • Prova att lägga till flera LED-lampor och programmera dem att blinka i sekvens.
cpp
/*****************************************************
* Laboration 1 - Blinka två lysdioder - övn 2
* 
* Programmet/sketchen går ut på att gul och röd 
* lysdiod blinkar en i taget en halv sekund.
******************************************************/

// Globala variabler.
int PIN_LED_RED = 13;     //Tilldelning av röd lysdiod i stiftet 13.
int PIN_LED_YELLOW = 12;  //Tilldelning av gul lysdiod i stiftet 12.

// variabel
int timer = 500;

void setup()
{
  //funktionen pinMode() anropas för att ställa in lysdioderna som UTSIGNALER.
  pinMode(PIN_LED_RED, OUTPUT);
  pinMode(PIN_LED_YELLOW, OUTPUT);
}


void loop()
{ 
  digitalWrite(PIN_LED_RED, HIGH);      //Röd lysdioden tänds 
  delay(timer);                         //väntar en halv sekund. 
  digitalWrite(PIN_LED_RED, LOW);       //Röd lysdioden släcks.
  delay(timer);                         //väntar en halv sekund.

  digitalWrite(PIN_LED_YELLOW, HIGH);   //Gul lysdioden tänds 
  delay(timer);                         //väntar en halv sekund. 
  digitalWrite(PIN_LED_YELLOW, LOW);    //Gul lysdioden släcks.
  delay(timer);                         //väntar en halv sekund.
}

Steg 7: Diskutera och Dokumentera ​

  • med dina klasskamrater eller lärare om vad som händer i koden och kretsen.
  • Dokumentera vad du har lärt dig i din laborationsrapport, inklusive ändringar du gjorde och deras effekter.

Sammanfattning ​

Denna laboration visade:

  • hur man kan simulera ett grundläggande Arduino-projekt i Tinkercad.
  • Du har lärt dig hur man kopplar en LED till en Arduino och
  • programmerar den för att blinka med en viss intervall.

Genom att använda Tinkercad har du fått en visuell och interaktiv förståelse för hur elektronik och programmering kan samverka i ett verkligt projekt.
