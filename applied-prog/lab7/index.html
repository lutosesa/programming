Laboration: Servo Motor ​

Inledning ​

I detta labb kommer vi att koppla till Arduino Uno en servo-motor och en potentiometer som kommer att styra servo-motorn. Genom att vrida potentiometern kan användaren ändra servo-motorns vinkel mellan 0° och 180°.

Sedan kommer vi att koppla in en fototransistor och tre lysdioder. Fototransistorn använder vi för att mäta ljusintensiteten och styra servo-motorn och lysdioderna (grön, gul och röd) baserat på ljusnivån.

Syfte med labbet ​

Du kommer att lära dig om:

  • Hur man använder biblioteket Servo och dess funktioner.
  • hur man kopplar in en servo-motor, potentiometer, fototransistor och lysdioder.
  • hur man kan styra dessa elektroniska komponenter genom att koda fram en algoritm på utvecklingsmiljön Arduino IDE.
  • hur används de huvudfunktionerna, funktioner som ingår i programmet och skapa och anropa funktioner.

Vad är Servo Motor? ​

En servo-motor i Arduino är en liten motor som kan rotera eller vrida sig till en specifik vinkel. Det är en typ av motor som är exakt och kan styras för att röra sig till bestämda positioner, vanligtvis inom ett intervall från 0° till 180°.

Hur fungerar en servo-motor med Arduino? ​

  • En servo-motor har tre kablar: en för ström (röd), en för jord (svart eller brun), och en styrsignal (gul eller vit).
  • Med hjälp av Arduino kan du skicka signaler till servo-motorn för att bestämma hur mycket den ska vrida sig, till exempel 45°, 90°, 135° eller 180°.
  • Du styr motorn genom att använda ett bibliotek som heter Servo.h, som gör det enkelt att ge kommandon till motorn.

Servo-motorer används ofta för att styra saker som robotarmar, dörrar eller andra rörliga delar i projekt som kräver precision.

Vad är en fototransistor? ​

En fototransistor i Arduino är en sensor som mäter ljus. Den fungerar ungefär som ett öga för Arduino, eftersom den känner av hur mycket ljus som faller på den.

Hur fungerar en fototransistor? ​

  • Fototransistorn släpper igenom mer ström när det finns mycket ljus och mindre ström när det är mörkt.
  • Den är ansluten till en analog pin på Arduino (t.ex. A0), som kan läsa av ljusnivån som ett värde. Detta värde kan variera beroende på hur starkt ljuset är.

I ett Arduino-projekt kan du använda en fototransistor för att göra något beroende på ljuset, som att tända en lampa när det blir mörkt eller styra en motor när ljuset ändras.

Övning 1 ​

Vi ska använda en potentiometer för att styra en servo-motor. Genom att vrida potentiometern kan användaren ändra servo-motorns vinkel mellan 0° och 180°.

Kopplings ritning ​

Programkod ​

cpp
/*************************************************
* Laboration 7 - Övning 1 - Servo-Motor
* Potentiometer styr servo-motor rotation
**************************************************/
 // bibliotek styr servo-motor med dess funktioner.
#include <Servo.h>

Servo servoPin9;

void setup()
{
    Serial.begin(9600);

    servoPin9.attach(9);
}

void loop()
{
    // map-funktionen omvandlar signaler från 
    // potentiometer 0-1023 till 0-180 grader.
    int angle = map(analogRead(A1), 0, 1023, 0, 180);

    Serial.print("Angle = ");
    Serial.println(angle);

    servoPin9.write(angle);
    delay(10);
}

Hur programmet fungerar:

  1. Läsning av potentiometer: Potentiometern är kopplad till den analoga ingången A1. Programmet läser av potentiometerns värde, vilket är ett analogt värde mellan 0 och 1023.
  2. Omvandling till vinkel: Med funktionen map() omvandlas potentiometerns värde (0–1023) till en motsvarande vinkel för servo-motorn (0–180 grader).
  3. Styrning av servo-motor: Servo-motorn roteras till den vinkel som beräknats från potentiometerns position.
  4. Seriell utskrift: Vinkeln skrivs ut till seriellmonitorn så att användaren kan övervaka den aktuella servovinkeln.

Övning 2 ​

Vi ska ansluta en fototransistor (ljussensor), 3 lysdioder och 3 resistorer. Med fototransistor som är ljusberoende kommer vi att kunna styra servo-motor och lysdioderna. Om fototransistor får mer ljus kommer servo-motor att rotera några grader till tills det kommer till 180 graders, samt de olika ljusdioder kommer att tändas beroende på värdet som fototransistorn avger.

Viktigt! På Tinkercad måste vi simulera ljusnivåerna genom att slumpa fram olika värde (ljusnivåer) för att skapa en dynamisk miljö där ljuset (slumpade värde) varierar.

Kopplings ritning ​

Programkod ​

cpp
/****************************************************
* Laboration 7 - Övning 2 - Servo motor
*
* Fototransistor som är ljusberoende styr och  
* kontrollerar både servo-motor och lysdioderna.
*****************************************************/

// Bibliotek <Servo.h> används för dess funktioner i servo-motorn.
#include <Servo.h>

// med detta objekt kan man anropa en del funktioner från Servo.h
Servo servoPin9;

const int LED_GREEN  = 6;
const int LED_YELLOW = 5;
const int LED_RED    = 4;

void setup()
{
  Serial.begin(9600);
  
  servoPin9.attach(9);
  
  pinMode(LED_GREEN,  OUTPUT);
  pinMode(LED_YELLOW, OUTPUT);
  pinMode(LED_RED,    OUTPUT);
}

void loop()
{
  //int sensor = analogRead(A0);
  
  // Skapar ett slumpmässigt värde mellan 0 och 50 för att simulera olika ljusnivå
  int sensor = random(0, 50); 
  
  // visar värdet på sensorn på monitor(konsol-fönstret).
  Serial.print("sensor = ");
  Serial.println(sensor);
  
  // om sensoren är mindre än 15 tänds den gröna lysdioden  
  // och motor roterar 20 grader.
  if(sensor < 15 ) // låg
  {
    digitalWrite(LED_GREEN,  HIGH);
    digitalWrite(LED_YELLOW, LOW);
    digitalWrite(LED_RED,    LOW);
    servoPin9.write(20);
    delay(1500);
  }
  
  // om sensoren är större än 14 och mindre än 26, 
  // tänds den gula lysdioden och motor roterar 90 grader.
  if(sensor > 14 && sensor < 26 ) // normal
  {
    digitalWrite(LED_GREEN,  LOW);
    digitalWrite(LED_YELLOW, HIGH);
    digitalWrite(LED_RED,    LOW);
    servoPin9.write(90);
    delay(1500);
  }
  
  // om sensoren är större än 25 tänds den röda lysdioden 
  // och motor roterar till 150 grader.
  if(sensor > 25 ) // hög
  {
    digitalWrite(LED_GREEN,  LOW);
    digitalWrite(LED_YELLOW, LOW);
    digitalWrite(LED_RED,    HIGH);
    servoPin9.write(150);
    delay(1500);
  }
}

Hur programmet fungerar:

  1. Läsning av ljusintensitet: Fototransistorn är ansluten till en analog ingång (A0), och dess värde mäts kontinuerligt.

  2. Styrning av servo-motorn: Beroende på ljusnivån (sensorvärdet) roteras servo-motorn till tre olika positioner (20°, 90°, eller 150°).

  3. Styrning av lysdioder: Tre lysdioder (grön, gul, röd) tänds beroende på ljusnivån:

    • Låg ljusnivå (< 15): Grön LED tänds, servo-motorn vrider sig till 20°.
    • Mellanljusnivå (15–25): Gul LED tänds, servo-motorn vrider sig till 90°.
    • Hög ljusnivå (> 25): Röd LED tänds, servo-motorn vrider sig till 150°.

  4. Simulera varierad ljusnivå:

    • Genom att lägga till ett slumpmässigt värde (ljusnivå) för sensor simuleras en dynamisk miljö där ljuset varierar.
