Att programmera med Arduino ​

Inledning ​

Att programmera med Arduino öppnar dörren till en värld där teknik och kreativitet möts. För gymnasieelever erbjuder Arduino en tillgänglig och engagerande väg in i både programmering och elektronik. Genom att skriva kod och se den omedelbara effekten på fysiska komponenter får du en praktisk och direkt upplevelse av hur programmering kan användas för att påverka den verkliga världen.

2. Vad är C++? ​

C++ är ett allmänt programmeringsspråk som används för att skapa allt från datorspel till operativsystem. Det är ett språk som är snabbt och effektivt, vilket gör det idealiskt för programmering nära hårdvaran, som med Arduino. När vi programmerar en Arduino, skriver vi kod i C++ som sedan laddas upp till mikrokontrollern på Arduino-kortet för att styra olika komponenter som LED-lampor, motorer och sensorer.

3. Minimalt program ​

Varje Arduino-program (kallas "sketch") består av två huvudfunktioner:

  • setup() : Körs en gång när programmet startar. Här initialiseras variabler, ställer in pin-lägen, etc.
  • loop() : Körs om och om igen tills programmet avslutas. Här sker huvudlogiken för ditt program.

Exempel

cpp
void setup()
{
   // det som skall köras vid start
}

void loop()
{
  // denna programloop körs i all
  // evighet, dvs så länge strömmen
  // är påslagen
}

4. Seriell monitor ​

Vi kommer att använda den seriella monitorn för att testa på exemplen som finns på detta avsnitt!

Den seriella monitorn i Arduino är ett inbyggt verktyg i Arduino IDE (Integrated Development Environment) som gör det möjligt för användaren att kommunicera med sitt Arduino-kort genom att skicka och ta emot textmeddelanden över en seriell anslutning (vanligtvis via USB).

Huvudfunktioner av seriella monitorn är följande
  1. Skicka data till Arduino: du kan skriva in text eller siffror i ett fält i seriella monitorn och skicka det till Arduino-kortet.
  2. Ta emot och visa data från Arduino: arduino-kortet kan skicka text eller numeriska data tillbaka till datorn, som sedan visas i den seriella monitorn.
  3. Felsökning: eftersom du kan se data som skickas från Arduino-kortet i realtid, är seriella monitorn ett kraftfullt verktyg för att felsöka din kod.
  4. Testning av programlogik: genom att använda seriella monitorn kan du testa hur ditt program reagerar på olika typer av inmatningar utan att behöva koppla upp extra hårdvara eller använda avancerade verktyg.

Exempel ​

Exempel på en ENKEL UTSKRIFT:

cpp
void setup() {
  Serial.begin(9600);  // Startar monitorn
  Serial.println("Hej på dig!");  // Skriver ut till monitorn
}

void loop() {
  // Här körs koden om och om igen!
}

8. Variabler och datatyper ​

En variabel är som en låda där du kan lagra data. I C++ måste du först deklarera vilken datatyp av data som ska lagras i variabeln.

8.1 Datatyper ​

De vanligaste datatyperna som används i C++ för Arduino är:

  • int : Heltal (t.ex. -2, 2, 7, 13)
  • float : Decimaltal (t.ex. 3.14, -0.001)
  • char : Enskilda tecken (t.ex. 'a', '9')
  • bool eller boolean : Sanna eller falska värden (true, false). Standarden i språket är bool, men arduino vill gärna att vi använder icke-standarden boolean.
  • byte : 8 bitar unsigned, dvs från 0 till 255
  • unsigned char : samma som byte, representerar 0 till 255
  • String : en samling av tecken som kan innehålla ord eller meningar. (t.ex "Hej Arduino")

Exempel på int, float, char, bool

cpp
// Här deklareras en int-variabel 'ledPin' och tilldelas värdet 13
int ledPin = 13;  

// Här deklareras en float-variabel 'temperatur' och tilldelas värdet 25.5
float temperatur = 25.5;  

// Här deklareras en char-variabel 'tecken' och tilldelas värdet A
char tecken = 'A';

// Här deklareras en bool-variabel 'ledstatus' och tilldelas värdet false
// En boolean som håller reda på om en LED är på eller av
bool ledStatus = true; 

//Här deklareras en String-variabel med tilldelning "Hej Arduino!"
String halsning = "Hej Arduino!";

{
    Serial.begin(9600);  // Startar monitorn
    
    Serial.println("ledPin = " + String(ledPin));
    Serial.println("temperatur = " + String(temperatur));
    Serial.println("tecken = " + String(tecken));
    Serial.println("ledStatus = " + String(ledStatus));
    Serial.println("halsning = " + halsning);
}

void loop() 
{
    // Tom loop eftersom vi bara demonstrerar en gång i setup()

9. Array ​

En array i Arduino är en samling av variabler av samma datatyp som lagras under ett och samma namn. Istället för att skapa flera enskilda variabler för att lagra flera värden, kan man använda en array för att effektivt hantera och komma åt dessa värden.

  • En array är som en lista av element (variabler/värde), där varje element har en specifik plats, eller index.
  • Alla element (värden) i arrayen har samma datatyp, såsom int, float, char.
  • Indexet för arrayen börjar alltid från 0, vilket betyder att det första elementet i arrayen finns på position 0.

Exempel på int, float, char, bool

cpp
// Deklarera och initiera en array med pinnar för tre LED-lampor
//      index =>  0   1   2   
int ledPins[3] = {7, 10, 13}; 

void setup() 
{
    Serial.begin(9600); // Startar monitorn

    Serial.println("ledPins[0] = " + String(ledPins[0]));
    Serial.println("ledPins[1] = " + String(ledPins[1]));
    Serial.println("ledPins[2] = " + String(ledPins[2]));
}

void loop() 
{  
  // Tom loop eftersom vi bara demonstrerar en gång i setup()  
}

while () ​

Ibland vill man att något skall upprepas sålänge något är sant. While utför det inom klammer i en evig loop sålänge uttrycket bakom while är sant.

I detta exempel väntar man på att användaren skriver ett visst antal tecken i den seriella monitorn innan man avslutar loopen

cpp
void setup() {
  Serial.begin(9600);  // Startar seriell kommunikation
  Serial.println("Skriv något i den seriella monitorn. \nNär du har skrivit 10 tecken, avslutas loopen.");

  int charCount = 0;  // Variabel för att hålla räkningen av tecken

  while (charCount < 10) {
    if (Serial.available() > 0) {
      char incomingChar = Serial.read();  // Läs det inkommande tecknet
      Serial.print("Du skrev: ");
      Serial.println(incomingChar);
      charCount++;  // Öka teckenräknaren
    }
  }

  Serial.println("Du har skrivit 10 tecken, loopen är klar.");
}

void loop() {
  // Tom loop eftersom vi bara demonstrerar en gång i setup()
}

do ... while () ​

I detta exempel använder man en do-while-loop för att läsa tecken från den seriella monitorn och upprepa utskriften tills användaren skriver bokstaven 'x'. Eftersom det är en do-while-loop, kommer den att köras minst en gång.

Exempel

cpp
void setup() {
  Serial.begin(9600);  // Startar seriell kommunikation
  Serial.println("Skriv tecken i den seriella monitorn. \nSkriv 'x' för att avsluta loopen.");

  char incomingChar;
  do {
    if (Serial.available() > 0) {
      incomingChar = Serial.read();  // Läs det inkommande tecknet
      Serial.print("Du skrev: ");
      Serial.println(incomingChar);
    }
  } while (incomingChar != 'x');  // Fortsätt loopen tills 'x' skrivs

  Serial.println("Du skrev 'x', loopen är klar.");
}

void loop() {
  // Tom loop eftersom vi bara demonstrerar en gång i setup()

21. digitalWrite(), digitalRead(), analogWrite(), analogRead() ​

Dessa funktioner används för att läsa från och skriva till pins på Arduino-kortet.

int sensorValue = analogRead(A0);  // Läs ett värde från den analoga pin A0

  // Kolla om det finns data att läsa
  if (Serial.available() > 0) {
    // Läs in en byte från den seriella bufferten

    Serial.println(incomingByte);

Slutsats ​

Att programmera med Arduino erbjuder gymnasieelever en praktisk och spännande introduktion till elektronik och programmering. Genom att kombinera hårdvara och mjukvara får man en djupare förståelse för hur datorer kan interagera med den fysiska världen. Arduino-labbar, som att tända LED-lampor, läsa sensordata eller styra motorer, gör att abstrakta programmeringsbegrepp blir konkreta och lättare att förstå.

Arduino-plattformen är både flexibel och användarvänlig, vilket gör den idealisk för nybörjare som vill lära sig grunderna i programmering (C++) och elektronik. Samtidigt ger den tillräckligt med djup för att mer avancerade elever ska kunna utmanas och utveckla komplexa projekt. Genom att arbeta med Arduino lär sig man inte bara om teknik, utan också om problemlösning, kreativitet och logiskt tänkande – färdigheter som är värdefulla oavsett vilken karriärväg man senare väljer att följa.

Sammanfattningsvis erbjuder Arduino en plattform där teori möter praktik, och där gymnasieelever kan ta sina första steg mot att bli framtidens innovatörer och ingenjörer.
