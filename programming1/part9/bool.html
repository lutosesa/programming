Python booleans ​

Booleans (bool) är en av de grundläggande datatyperna i Python. De används för att representera sanna (True) eller falska (False) värden. I programmering är booleans ofta involverade i villkorssatser och kontrollflöden, såsom if-satser, while-loopar och liknande.

1. Booleska värden ​

  • I programmering behöver du ofta veta om ett uttryck är True eller False.
  • Du kan utvärdera vilket uttryck som helst i Python och få ett av två svar, True eller False.
  • När du jämför två värden utvärderas uttrycket och Python returnerar det booleska svaret:

Exempel

python
print(10 > 9)    # True
print(10 == 9)   # False
print(10 < 9)    # False

2. Booleans i villkorssatser ​

Booleans används ofta i villkorssatser för att styra flödet av programmet. Här är ett exempel med en if-sats:

Exempel

python
is_raining = True

if is_raining:
    print("Ta med ett paraply!")
else:
    print("Ingen behov av paraply.")

3. Logiska Operatorer ​

Python tillhandahåller flera logiska operatorer för att kombinera och jämföra booleanvärden:

3.1 and-operator ​

and returnerar True om båda operanderna är True, annars False.

python
print(True and True)      # Output: True
print(True and False)     # Output: False
print(False and False)    # Output: False

3.2 or-operator ​

or returnerar True om minst en operand är True, annars False.

python
print(True or True)      # Output: True
print(True or False)     # Output: True
print(False or False)    # Output: False

3.3 not-operator ​

not returnerar det motsatta booleanvärdet.

python
print(not True)     # Output: False
print(not False)    # Output: True

5. Användning av Booleans i Funktioner ​

Booleans används ofta i funktioner för att kontrollera tillstånd och besluta om vissa åtgärder ska vidtas.

Exempel: funktionen kontrollerar om talet (parametern number) är jämt.

python
def is_even(number):  
    return number % 2 == 0

print(is_even(4))    # Output: True
print(is_even(7))    # Output: False

6. Sanningsvärden av olika datatyper ​

  • Funktionen bool() låter dig utvärdera vilket värde som helst och returnera True eller False.
  • Nästan vilket värde som helst utvärderas True om det har något slags innehåll.
  • Vilken sträng som helst är True, förutom tomma strängar.
  • Vilket nummer som helst är True, utom 0.
  • Alla listor, tuple, set och dictionary är True, förutom tomma.

Exempel:

python
# Följande kommer att returnera True:
print(bool(1))           # Output: True
print(bool([1, 2, 3]))   # Output: True (icke-tom lista)
print(bool("Hello"))     # Output: True (icke-tom sträng)

# Följande kommer att returnera False:
print(bool(0))           # Output: False
print(bool(""))          # Output: False (tom sträng)
print(bool([]))          # Output: False (tom lista)
print(bool(()))          # Output: False (tom tuple)
print(bool({}))          # Output: False (tom set)
print(bool(None))        # Output: False
print(bool(False))       # Output: False

Slutsats ​

Booleans är en viktig del av programmering i Python och används i många olika sammanhang. De spelar en nyckelroll i villkorssatser, loopar och logiska operationer. Genom att förstå och använda booleans effektivt kan du skapa mer komplexa och dynamiska program.
