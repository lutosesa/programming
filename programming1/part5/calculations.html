Python Beräkningar ​

Python är ett kraftfullt och lättanvänt programmeringsspråk som gör det enkelt att utföra olika typer av beräkningar. I denna del kommer vi att gå igenom grundläggande matematiska operationer, aritmetiska operationer, användning av inbyggda Python-funktioner, samt introducera lite mer avancerade begrepp som hantering av operatorer och användning av biblioteket math.

1. Grundläggande Aritmetik ​

Python tillåter dig att utföra de vanligaste aritmetiska operationerna:

  • Addition (+): Lägger till två tal.
  • Subtraktion (-): Subtraherar ett tal från ett annat.
  • Multiplikation (*): Multiplicerar två tal.
  • Division (/): Dividerar ett tal med ett annat. Resultatet är alltid ett flyttal.
  • Exponentiering ()**: Höjer ett tal till en viss exponent.
  • Modulus (%): Ger resten när ett tal delas med ett annat.
  • Heltalsdivision (//): Dividerar två tal och returnerar endast heltalsdelen av resultatet.

Exempel

python
# Addition
print(5 + 3)  # Output: 8

# Subtraktion
print(10 - 4)  # Output: 6

# Multiplikation
print(7 * 2)  # Output: 14

# Division
print(15 / 3)  # Output: 5.0

# Exponentiering
print(2 ** 3)  # Output: 8

# Modulus
print(10 % 3)  # Output: 1

# Heltalsdivision
print(10 // 3)  # Output: 3

2. Datatyper för Beräkningar ​

I Python är de vanligaste datatyperna för numeriska beräkningar int och float.

  • Heltal (int): Representerar heltal, t.ex. 1, 2, 3.
  • Flyttal (float): Representerar decimaltal, t.ex. 1.5, 3.14, 0.001. När du utför beräkningar med dessa datatyper är det viktigt att vara medveten om vilken typ av resultat du förväntar dig.

Exempel

python
# Heltalsberäkning
a = 10
b = 3
resultat = a + b  # resultat är 13 (int)
print(resultat)

# Decimaltalsberäkning
x = 10.5
y = 4.2
resultat = x * y  # resultat är 44.1 (float)
print(resultat)

# Blandad beräkning (int och float)
m = 5
n = 2.5
resultat = m + n  # resultat är 7.5 (float)
print(resultat)

3. Använda Variabler ​

I Python kan vi använda variabler för att lagra data och sedan använda dessa variabler i beräkningar. Variabler gör din kod mer läsbar och lättare att underhålla.

Exempel

python
# Definiera variabler
a = 5
b = 7

# Utför beräkningar med variabler
summa = a + b
produkten = a * b

# Skriv ut resultaten
print("Summan är:", summa)  # Output: Summan är: 12
print("Produkten är:", produkten)  # Output: Produkten är: 35

4. Matematiska Funktioner och Modulen math ​

Python har ett inbyggt bibliotek som heter math som innehåller många användbara matematiska funktioner. Här är några vanliga funktioner:

  • Kvadratroten av ett tal: math.sqrt(x)
  • Potensberäkning (exponentiering): math.pow(x, y)
  • Talet PI: math.pi
  • Avrundning neråt: math.floor(x)
  • Avrundning uppåt: math.ceil(x)
  • Minimum och Maximum: min(x, y, ...) och max(x, y, ...)

För att använda math-modulen måste vi importera den:

Exempel

python
import math

# Kvadratrot
print(math.sqrt(16))  # Output: 4.0

#Potensberäkning
print(math.pow(2, 3))  # resultat blir 8.0

#Talet pi
print(math.pi)  # resultat blir 3.141592653589793

#Avrundning neråt
print(math.floor(3.7))  # resultat blir 3

#Avrundning uppåt
print(math.ceil(3.1))  # resultat blir 4

# Minimum och Maximum
minst  = min(3, 1, 4, 2)  # resultat blir 1
storst = max(3, 1, 4, 2)  # resultat blir 4
print(minst, storst)

5. Operatorer och Prioritering ​

I Python, precis som i matematik, har operatorer en viss ordning som de utvärderas i:

  1. Parenteser: Beräkningar inuti parenteser utförs först.
  2. Exponentiering: Utförs efter parenteser.
  3. Multiplikation och division: Utförs efter exponentiering.
  4. Addition och subtraktion: Utförs sist.

Du kan använda parenteser för att styra ordningen på beräkningarna.

Exempel

python
resultat = 10 + 2 * 3  # Multiplikation först, sedan addition, resultat = 16
print(resultat)

resultat = (10 + 2) * 3  # Parenteser först, sedan multiplikation, resultat = 36
print(resultat)

6. Interaktiva Beräkningar ​

Python kan ta emot användarinmatning med funktionen input(). Denna funktion returnerar alltid en sträng, så om du vill använda inmatningen i beräkningar måste du omvandla strängen till ett int eller float.

Exempel

python
# Ta emot ett tal från användaren
tal1 = float(input("Ange det första talet: "))
tal2 = float(input("Ange det andra talet: "))

# Utför en beräkning
summa = tal1 + tal2

# Skriv ut resultatet
print(f"Summan av {tal1} och {tal2} är {summa}")

7. Exempelprojekt: Enkel kalkylator ​

Låt oss nu tillämpa vad vi lärt oss genom att skapa en enkel kalkylator som kan utföra olika aritmetiska operationer baserat på användarens val.

Exempel

python
print("---------------------------")
print(" ENKEL KALKYLATOR")
print("---------------------------")

# Ta emot två tal från användaren
tal1 = float(input("<Ange det första talet> "))
tal2 = float(input("<Ange det andra talet> "))

# Fråga användaren vilken operation som ska utföras
operation = input("<Välj en operation (+, -, *, /, **, //, %)> ")


# Utför operationen
if operation == "+":
    resultat = tal1 + tal2
elif operation == "-":
    resultat = tal1 - tal2
elif operation == "*":
    resultat = tal1 * tal2
elif operation == "/":
    resultat = tal1 / tal2
elif operation == "**":
    resultat = tal1 ** tal2
elif operation == "//":
    resultat = tal1 // tal2
elif operation == "%":
    resultat = tal1 % tal2
else:
    resultat = "Ogiltig operation"

# Skriv ut resultatet
print(f"\nResultatet är: {resultat}")
