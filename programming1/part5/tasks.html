Uppgifter om datatyper ​

Uppgifterna fokuserar på Python-datatyperna int, float, string och bool. Dessa uppgifter kommer att hjälpa dig att förstå och använda dessa datatyper.

Uppgift 1: Summera och beräkna medelvärdet av tre tal ​

Datatyper: int, float

  1. Skriv ett program som tar tre tal (kan vara heltal eller flyttal) som input från användaren.
  2. Programmet ska beräkna summan av talen samt medelvärdet av dem.
  3. Skriv ut både summan och medelvärdet.

Uppgift 2: Beräkna omkrets och area på en cirkel ​

Datatyper: int, float

  1. Skriv ett program som ber användaren mata in radien på en cirkel.
  2. Programmet ska sedan beräkna omkretsen och arean på cirkeln med följande formler:
    • Omkrets=2π∗radie
    • Area=π∗radie2
  3. Använd float för att hantera radien och beräkningarna.
  4. Glöm inte att importera modulen math för pi-talet, math.pi
python
import math

radie = float(input("Ange radien på cirkeln: "))
omkrets = 2 * math.pi * radie

Uppgift 3: Omvandla Celsius till Fahrenheit ​

Datatyp: float

  1. Skriv ett program som tar en temperatur i Celsius som input.
  2. Programmet ska omvandla temperaturen till Fahrenheit med följande formel:
    • Fahrenheit=(Celsius∗9/5)+32
  3. Skriv ut resultatet med två decimaler.

Uppgift 4: Hantera personens fullständiga namn ​

Datatyp: str

  1. Skriv ett program som frågar användaren om deras förnamn och efternamn.
  2. Skriv ut en hälsning med deras fullständiga namn.

Uppgift 5: Beräkna en cirkels diameter och volymen på en sfär ​

Datatyper: int, float

  1. Skriv ett program som ber användaren mata in radien på en cirkel (eller sfär).
  2. Programmet ska beräkna cirkelns diameter och volymen på en sfär med den radien
    • Diameter=2∗radie
    • Volym=43π∗radie3
  3. Skriv ut resultaten

Uppgift 6: Sanningsvärden och logiska operationer ​

Datatyp: bool

  1. Skriv ett program som tar två heltal som input från användaren.
  2. Programmet ska sedan skapa tre booleska variabler som representerar om:
    • Det första talet är större än det andra.
    • Det första talet är jämnt.
  3. Skriv ut resultaten av de två jämförelserna.
