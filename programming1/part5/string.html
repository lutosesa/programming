Python string ​

Strings i Python är sekvenser av tecken som används för att lagra och manipulera text. Här är en grundlig genomgång av Python-strings, från grundläggande koncept till mer avancerade tekniker.

1. Skapa och skriva ut strings ​

För att skapa en string använder du antingen enkla (') eller dubbla (") citationstecken.

python
str1 = 'Hello'
str2 = "World"

print(str1)
print(str2)

2. String-indexering och slicing ​

Strings är indexerade från 0. Du kan få tillgång till enskilda tecken genom indexering och delar av strängen genom slicing.

python
str1 = "Hello World!"

# Indexering
print(str1[0])   # Output: H
print(str1[-1])  # Output: !

# Slicing
print(str1[0:5])  # Output: Hello
print(str1[6:11]) # Output: World

4. Formatering av strings ​

Det finns flera sätt att formatera strängar i Python.

4.1. f-strings (formaterade strängar) ​

F-String introducerades i Python 3.6 och är nu det föredragna sättet att formatera strängar.
För att ange en sträng som en f-sträng, sätt helt enkelt en f framför strängen literal och lägg till klammerparenteser eller måsvingar {} som platshållare för variabler och andra operationer.

Platshållare {} ​

  • En platshållare kan innehålla variabler, operationer, funktioner och modifierare för att formatera värdet.
python
name = "Johnny"
age = 16
text = f"Mitt namn är {name} och jag är {age} år."
print(text)
1
2
3
4
  • En platshållare kan innehålla Python-kod, som matematiska operationer:
python
txt = f"Priset är {25 * 4} kronor"
print(txt)
1
2

Modifierare ​

  • En platshållare kan inkludera en modifierare för att formatera värdet.
  • En modifierare inkluderas genom att lägga till ett kolon : följt av en formateringstyp, som .2f till exempel betyder ett fast punktnummer med 2 decimaler:
python
# Visa priset med 2 decimaler:

price = 25
txt = f"Priset är {price:.2f} Kronor"
print(txt)
1
2
3
4
5

Här har du andra modifierare som är användbara:

Bredd och justering:

  • *:<n : Vänsterjustering i ett fält av bredd n.
  • :>n : Högerjustering i ett fält av bredd n.
  • :^n : Centrering i ett fält av bredd n.
python
value = 42
print(f"{value:<5}")  # '42   '
print(f"{value:>5}")  # '   42'
print(f"{value:^5}")  # ' 42 '

Utfyllnad:

  • :0n : Fyller med nollor fram till bredd n.
  • Exempel: value = 42, print(f"{value:05}") ger 00042.
python
value = 42
print(f"{value:05}") # ger 00042.

9.2. String interpolation ​

Python 3.8 introducerade en ny metod för att sätta in värden i strängar med hjälp av =.

python
x = 10
y = 20
print(f'{x=} och {y=}')  # Output: x=10 och y=20
1
2
3

Slutsats ​

  • Strängar i Python är sekvenser av tecken som används för att representera text.
  • De skapas genom att omge text med enkla ('...') eller dubbla ("...") citattecken.
  • Strängar är immutabla, vilket innebär att de inte kan ändras efter att de har skapats.
  • Python erbjuder ett brett utbud av inbyggda metoder för att manipulera och bearbeta strängar, inklusive metod för att ändra textens formatering, söka efter substrängar och dela upp strängar i listor.
  • Dessutom stödjer Python f-strängar (f"...") och metoden .format() för enkel och flexibel stränginterpolering och formatering.
  • Tack vare dessa egenskaper är strängar i Python kraftfulla och användarvänliga verktyg för textbearbetning.
