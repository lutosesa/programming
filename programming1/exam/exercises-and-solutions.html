1. Vad är en variabel och hur används den i Python? Ge ett exempel. ​


Lösning: ​

En variabel är en plats där data lagras och som kan användas senare i programmet. Variabeln får ett namn och ett värde, och detta värde kan ändras under programmets gång.

Exempel

python
namn = "Kalle Anka"
ålder = 17

2. Vad är en kommentar i Python och varför är det bra att använda dem? ​


Lösning: ​

En kommentar är en förklaring i koden som Python ignorerar när programmet körs. Kommentarer används för att göra koden lättare att förstå, både för dig själv och för andra.

Exempel

python
# Detta är en kommentar

3. Vilka datatyper finns det i Python? Ge ett exempel på var och en. ​


Lösning: ​

En variabel är en plats där data lagras och som kan användas senare i programmet. Variabeln får ett namn och ett värde, och detta värde kan ändras under programmets gång.

  • Sträng (str): Används för text.
    Exempel: namn = "Kalle"

  • Heltal (int): Används för hela tal.
    Exempel: ålder = 17

  • Flyttal (float): Används för decimaltal.
    Exempel: pris = 19.99

  • Boolean (bool): Används för sant eller falskt.
    Exempel: är_myndig = True

8. Vad är en while-loop och hur fungerar den? Skriv ett exempel där en while-loop räknar från 1 till 5. ​


Lösning: ​

En while-loop upprepar koden så länge ett specifikt villkor är sant.

Exempel

python
x = 1
while x <= 5:
    print(x)
    x += 1

9. Hur kan du bryta en loop i förtid? Ge ett exempel. ​


Lösning: ​

Med break-satsen kan du avbryta en loop tidigare än planerat.

Exempel

python
x = 1
while x <= 10:
    if x == 5:
        break       # Avbryter loopen när x är 5
    print(x)
    x += 1

  • Syntaxfel: Uppstår när koden inte följer Pythons syntaxregler, t.ex. glömma ett kolon.

    Exempel

    python

print("x är 10")

    Exempel

    python

12. Vad gör continue i en loop? Skriv ett program där loopen hoppar över ett specifikt värde. ​


Lösning: ​

continue gör att resten av koden i loopen hoppas över för den aktuella iterationen, men loopen fortsätter med nästa värde.

Exempel

python
x = 0
while x < 5:
    x += 1
    if x == 3:
        continue    # Hoppar över utskriften när x är 3
    print(x)

13. Vad innebär typkonvertering? Varför är det viktigt? ​


Lösning: ​

Typkonvertering innebär att man ändrar datatypen på ett värde, till exempel från en sträng till ett heltal. Det är viktigt eftersom vissa operationer kräver specifika datatyper, som matematiska beräkningar som endast fungerar med tal.

Exempel

python
siffra = "5"
siffra = int(siffra)  # Omvandlar strängen till ett heltal

14. Vad är break och continue i loopar och hur skiljer de sig? ​


Lösning: ​

  • break: Avslutar loopen helt.
  • continue: Hoppar över resten av koden i loopen för den aktuella iterationen och fortsätter med nästa.
