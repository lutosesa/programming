Grund begrepp ​

Denna del är en sammanfattning som täcker första delen av de grundläggande begreppen inom programmering 1. Här går vi igenom viktiga begrepp såsom variabler, kommentarer, datatyper, in- och utmatning, beräkningar, typkonverteringar, villkorssatser, loopar och felsökning. Varje avsnitt innehåller korta definitioner och exempel för att ge en tydlig förståelse av hur dessa begrepp används i praktiken.

Genom att studera dessa grunder och öva på de tillhörande frågorna kommer du vara väl förberedd inför provet.

1. Variabler ​

En variabel i Python är som en namnlapp som du sätter på något du vill spara och använda senare i programmet. Det kan vara ett namn, en siffra, eller något annat. När du skapar en variabel, ger du den ett namn och ett värde.

Exempel:

python
namn = "Kalle Anka"
ålder = 17
1
2

I exemplet ovan har vi två variabler:

  • namn som sparar texten "Kalle".
  • ålder som sparar siffran 17.

Du kan använda dessa variabler senare i programmet för att göra saker med dem, som att skriva ut.

2. Kommentarer ​

Kommentarer i Python är texter i koden som inte körs av datorn. De används för att skriva förklaringar eller anteckningar i koden så att du eller andra lättare kan förstå vad som händer. Kommentarer börjar med ett #-tecken och allt som skrivs efter det blir en kommentar.

Exempel:

python
# Detta är en kommentar
namn = "Kajsa Anka"  # Denna rad sparar namnet "Sara" i variabeln
1
2

I detta exempel kommer Python att ignorera texten efter # och bara köra koden. Kommentarer är bra för att göra koden lättare att läsa!

3. Datatyper ​

Datatyper i Python beskriver vilken typ av information som en variabel innehåller. Olika datatyper används för olika sorters data, som text, siffror eller sant/falskt-värden. Här är några vanliga datatyper i Python:

  1. Sträng (str): Används för text. Den skrivs inom citationstecken, som "Hej".

Exempel:

python
namn = "Kalle"
1
  1. Heltal (int): Används för hela nummer, som 5 eller 100.

Exempel:

python
ålder = 17
1
  1. Flyttal (float): Används för decimaltal, som 3.14 eller 10.5.

Exempel:

python
pris = 19.99
1
  1. Boolean: Används för sant eller falskt, som True (sant) eller *False (falskt)

Exempel:

python
är_myndig = True
1

4. Input ​

input i Python är ett sätt att ta in information från användaren. När du använder input, ber du användaren att skriva något, och programmet sparar det de skriver. Det kan vara allt från ett namn till ett tal.

Exempel:

python
namn = input("Vad heter du? ")
1

I detta exempel kommer programmet att fråga användaren "Vad heter du?". Det användaren skriver in sparas i variabeln namn, och kan användas senare i programmet.

Viktigt: Allt som användaren skriver in via input sparas som en text (sträng), även om det ser ut som ett tal. Vill du ha det som ett tal, måste du omvandla det.

5. Output - print() ​

I Python används output för att visa information på skärmen med hjälp av funktionen print(). Det finns olika sätt att skriva ut text och variabler i Python. Här går vi igenom tre vanliga sätt: utskrift med komma-tecken, f-strängar och plus-operatorn.

5.1 Utskrift med komma-tecken ​

När du använder komma-tecken kan du skriva ut flera värden tillsammans. Python lägger automatiskt till ett mellanslag mellan varje värde.

Exempel:

python
namn = "Kalle"
ålder = 17
print("Namn:", namn, "Ålder:", ålder)
1
2
3

Förklaring: Här skriver vi ut texten "Namn:", variabeln namn, texten "Ålder:" och variabeln ålder. Python lägger automatiskt ett mellanslag mellan dessa värden.

Utskrift:

python
Namn: Kalle Ålder: 17

5.2 f-strängar ​

Med f-strängar kan du enkelt skriva ut text och variabler tillsammans i en sträng. Du sätter ett f framför strängen och placerar variablerna inom { } inuti strängen.

Exempel:

python
namn = "Kalle"
ålder = 17
print(f"Namn: {namn} , Ålder: {ålder}")
1
2
3

Förklaring: Inuti { } ersätts variabeln med dess värde direkt i strängen.

Utskrift:

python
Namn: Kalle , Ålder: 17

5.3 Plus-operatorn ​

Plus-operatorn ( + ) kan användas för att slå ihop (konkatenera) strängar. Men kom ihåg att du bara kan använda + för att kombinera strängar, inte siffror direkt.

Exempel:

python
namn = "Kalle"
print("Hej " + namn + "!")
1
2

Förklaring: Här slår vi ihop flera strängar med +. Mellanslag måste skrivas ut manuellt genom att lägga till dem i strängarna.

Utskrift:

python
Hej Kalle!

6. Beräkningar ​

Beräkningar i Python innebär att man gör matematiska operationer, som addition (plus), subtraktion (minus), multiplikation (gånger) och division (delat). Python kan användas som en miniräknare för att räkna ut resultat.

6.1 Vanliga matematiska operationer: ​

  • Addition ( + ): Lägg till två tal.
    • Exempel: 3 + 2 ger 5
  • Subtraktion ( - ): Dra av ett tal från ett annat.
    • Exempel: 5 - 3 ger 2
  • Multiplikation ( * ): Multiplicera två tal.
    • Exempel: 4 * 2 ger 8
  • Division ( / ): Dela ett tal med ett annat.
    • Exempel: 10 / 2 ger 5.0

Exempel:

python
tal1 = 10
tal2 = 5
summa = tal1 + tal2
print(summa)    # Resultatet blir 15
1
2
3
4

Förklaring: Här sparas talen 10 och 5 i variablerna tal1 och tal2. Sedan beräknas deras summa och resultatet skrivs ut.

6.2 Interaktiva beräkningar: ​

Du kan också fråga användaren om tal och göra beräkningar med dem. Här behöver vi använda input och omvandla texten till ett tal.

Exempel:

python
tal = int(input("Skriv ett tal: "))  # Omvandlar text till heltal
resultat = tal + 10
print(f"Resultatet är: {resultat}")
1
2
3

Förklaring: Användaren skriver in ett tal, programmet lägger till 10, och sedan skrivs resultatet ut.

7. typkonverteringar ​

Typkonverteringar i Python innebär att man ändrar en datatyp till en annan, till exempel från en sträng (text) till ett heltal (nummer), eller tvärtom. Detta är viktigt när du behöver arbeta med olika sorters data i programmet.

  • Från sträng till heltal: Används när du har ett tal i textform och vill göra matematiska beräkningar med det.

    Exempel: Använd funktionen int() för att omvandla en sträng till ett heltal.

    python
    siffra = "5"
siffra = int(siffra)  # Nu är 'siffra' ett heltal, inte en text
    1
    2

  • Från sträng till flyttal: När du behöver omvandla en text till ett decimaltal.

    Exempel: Använd funktionen float().

    python
    tal = "3.14"
tal = float(tal)  # Nu är 'tal' ett flyttal
    1
    2

  • Från tal till sträng: Används när du vill slå ihop ett tal med en text för att skriva ut det.

    Exempel: Använd funktionen str() för att omvandla ett tal till en sträng.

    python
    ålder = 18
text = "Jag är " + str(ålder) + " år gammal."
print(text)  # Skriver: Jag är 18 år gammal.
    1
    2
    3

Varför är typkonvertering viktigt? ​

När du använder input() för att fråga användaren om ett tal får du alltid en sträng (text), även om det ser ut som ett tal. För att kunna räkna med detta måste du först omvandla det till ett heltal eller flyttal.

Exempel:

python
nummer = input("Skriv ett nummer: ")    # 'nummer' är en sträng
nummer = int(nummer)      # Nu är 'nummer' ett heltal
1
2

8. villkorssatser ​

Villkorssatser i Python används för att fatta beslut i programmet. De gör att programmet kan köra viss kod om ett villkor är sant, och annan kod om villkoret är falskt. Den vanligaste villkorssatsen är if, men det finns också else och elif som används för att lägga till fler alternativ.

Hur fungerar villkorssatser? ​

  • if-sats: Körs om ett villkor är sant.

  • else-sats: Körs om villkoret i if är falskt.

  • elif-sats: Används om du vill lägga till fler villkor mellan if och else.

    Exempel:

    python
    ålder = 18

if ålder >= 18:
    print("Du är myndig.")        # Körs om villkoret är sant
else:
    print("Du är inte myndig.")   # Körs om villkoret är falskt
    1
    2
    3
    4
    5
    6

    Förklaring:

    • Om värdet av ålder är 18 eller högre (sant villkor), skriver programmet ut "Du är myndig."
    • Om värdet av ålder är mindre än 18 (falskt villkor), skriver programmet ut "Du är inte myndig."

elif för flera alternativ: ​

Om du har fler än två alternativ, kan du använda elif (förkortning av "else if").

Exempel:

python
ålder = 18

if ålder >= 18:
    print("Du är myndig.")
elif ålder >= 15:
    print("Du är tonåring.")
else:
    print("Du är barn.")

Förklaring:

  • Om ålder är 18 eller högre, skrivs "Du är myndig."
  • Om ålder är minst 15 men mindre än 18, skrivs "Du är tonåring."
  • Om inget av dessa villkor är sant, skrivs "Du är barn."

Indentation (indrag): ​

Indrag (mellanslag) är viktigt i Python. Koden under en if, else, eller elif måste ha indrag, annars får du fel.

9. While-loopar ​

En while-loop upprepar en viss del av koden så länge ett visst villkor är sant. Det betyder att den fortsätter att köra tills villkoret blir falskt.

Exempel:

python
x = 1
while x <= 5:
    print(x)
    x += 1      # Öka x med 1 varje gång/varv
1
2
3
4

Förklaring:

  • Så länge x är mindre än eller lika med 5, kommer loopen att köra och skriva ut värdet av x. Varje gång ökas x med 1.
  • När x blir 6, avslutas loopen eftersom villkoret x <= 5 inte längre är sant.

Utskrift:

python
1
2
3
4
5

Break-sats: ​

Med break kan du avbryta en loop, även om villkoret fortfarande är sant. Det används om du vill avsluta loopen tidigare baserat på ett visst villkor.

Exempel:

python
x = 1
while x <= 10:
    if x == 5:
        break  # Avslutar loopen när x är 5
    print(x)
    x += 1
1
2
3
4
5
6

Förklaring:

  • När x blir 5, avslutas loopen direkt på grund av break.

Utskrift:

python
1
2
3
4

Continue-sats: ​

Med continue kan du hoppa över resten av koden i loopen och gå direkt till nästa varv. Det betyder att resten av koden i loopen för den omgången inte körs, men loopen fortsätter från nästa iteration.

Exempel:

python
x = 0
while x < 5:
    x += 1
    if x == 3:
        continue    # Hoppar över utskriften när x är 3
    print(x)

  • När x är 3, hoppar continue över utskriften och går direkt till nästa varv av loopen.

10. For-loopar ​

En for-loop används för att upprepa kod ett visst antal gånger, eller för att gå igenom varje element i en samling (som en lista eller en sträng). Du kan se det som ett sätt att säga "gör detta för varje sak i en lista" eller "gör detta så många gånger som jag bestämmer."

Exempel (gå igenom en lista):

python
frukter = ["äpple", "banan", "apelsin"]
for frukt in frukter:
    print(frukt)
1
2
3

Förklaring:

  • Loopen går igenom varje element i listan frukter och skriver ut varje frukt på en egen rad.

Utskrift:

python
äpple
banan
apelsin

Break-sats i for-loopar: ​

Med break kan du avbryta en for-loop i förtid. När break används, slutar loopen omedelbart, även om den inte har gått igenom alla element än.

Exempel

python
frukter = ["äpple", "banan", "apelsin"]
for frukt in frukter:
    if frukt == "banan":
        break           # Avslutar loopen när vi når "banan"
    print(frukt)
1
2
3

Förklaring:

  • När loopen når "banan", avslutas den på grund av break. Resten av listan ignoreras.

Utskrift:

python

Continue-sats i for-loopar: ​

Med continue kan du hoppa över en del av loopen och gå direkt till nästa varv. Resten av koden i loopen körs inte för det elementet, men loopen fortsätter med nästa.

Exempel:

python
frukter = ["äpple", "banan", "apelsin"]
for frukt in frukter:
    if frukt == "banan":
        continue          # Hoppar över "banan"
    print(frukt)

  • När loopen når "banan", hoppar den över utskriften för just den frukten och fortsätter med nästa frukt.

11. Felsökning av vanliga feltyper ​

Felsökning i Python handlar om att hitta och rätta till fel som gör att programmet inte fungerar som förväntat. Vanliga feltyper kan vara allt från stavfel i koden till att försöka använda en variabel på fel sätt. När ett fel uppstår, ger Python ett felmeddelande som hjälper dig att förstå vad som är fel och var i koden det hände.

Syntaxfel (SyntaxError): ​

Detta fel händer när koden inte följer Pythons regler. Exempel på syntaxfel kan vara att glömma ett kolon : i slutet av en if-sats eller felaktigt indrag (mellanslag).

Exempel:

python
if x == 10    # Här saknas ett kolon (SyntaxError)
    print("x är 10")
1
2

Felsökning: Kontrollera koden och se till att den följer rätt syntax, som att ha rätt kolon, parenteser och indrag.

Namnfel (NameError): ​

Detta fel händer när du försöker använda en variabel eller funktion som inte finns. Kanske har du glömt att definiera variabeln, eller så är namnet felstavat

Exempel:

python
print(namn)   # Om variabeln 'namn' inte har definierats (NameError)
1

Felsökning: Kontrollera om du har definierat alla variabler innan du använder dem och att du stavar dem rätt.

Typfel (TypeError): ​

Detta fel händer när du försöker använda fel typ av data på ett sätt som inte fungerar. Till exempel, du kanske försöker lägga ihop en sträng och ett tal, vilket inte är tillåtet.

Exempel:

python
# Du kan inte lägga ihop en sträng och ett heltal (TypeError)
print("Jag är " + 18)
1
2

Lösning: Omvandla datatyper när det behövs, till exempel med str() för att omvandla ett tal till en sträng.

python
print("Jag är " + str(18))  # Korrekt sätt
1

Värdefel (ValueError): ​

Detta fel händer när du försöker konvertera en datatyp, men värdet inte är kompatibelt. Till exempel att försöka omvandla text till ett heltal som inte är ett nummer.

Exempel:

python
 # Du kan inte omvandla texten "Hej" till ett tal (ValueError)
siffra = int("Hej")
1
2

Felsökning: Kontrollera att värdet är rätt för den datatyp du vill använda, till exempel att en sträng faktiskt innehåller ett nummer innan du omvandlar den.
