Python teminal (konsol-fönster) ​

Välkommen till en genomgång om hur man kan använda ANSI escape-sekvenser i Python-terminalen för att ändra textfärger, bakgrundsfärger och göra andra coola saker. Denna genomgång är riktad till gymnasiestudenter som är nyfikna på hur man kan förbättra sina terminalutskrifter med Python och skapa visuellt tilltalande textutmatningar.

1. Vad är ANSI escape-sekvenser? ​

ANSI escape-sekvenser är speciella teckensträngar som används för att styra textformattering på terminaler. Dessa sekvenser börjar med ESC-tecknet (Escape, ASCII-kod 27), följt av en sekvens av tecken som anger vad som ska göras, till exempel ändra färg eller flytta markören.

Exempel på en ANSI escape-sekvens:

  • \033[31m: Sätter textfärgen till röd.
  • \033[0m: Återställer textformatteringen till standard.

2. Använda ANSI escape-sekvenser i Python ​

För att använda ANSI escape-sekvenser i Python behöver vi använda print-funktionen och inkludera escape-sekvenserna i strängarna vi vill skriva ut.

Exempel

python
print("\033[31mDetta är röd text\033[0m")
1

Ovanstående kod kommer att skriva ut "Detta är röd text" i röd färg.

4. Tömma terminalen ​

Att tömma terminalen kan göras med os.system-kommandot. Kommandot är olika beroende på om du använder Windows eller ett Unix-baserat system (som macOS eller Linux).

Exempel

python
import os

def clear_console():
    os.system('cls' if os.name == 'nt' else 'clear')

# Använd funktionen
clear_console()
print("Terminalen är nu tömd!")
