Utmatning och inmatning ​

I Python är interaktionen med användaren via konsolen central för många applikationer. Två av de mest grundläggande funktionerna som möjliggör denna interaktion är print() och input().

1. Utmatning - print() ​

1.1 Grundläggande användning ​

'print()'-funktionen används för att skriva ut meddelanden, variabler eller andra objekt till konsolen. Det är en av de mest grundläggande och använda funktionerna i Python.

I detta exempel kommer det att skrivas ut texten "Hej, världen!" till konsolen.

python
print("Hej, världen!")
1

1.2 Skriva ut flera värden ​

Du kan skriva ut flera värden samtidigt genom att separera dem med kommatecken.

I detta exempel kommer det att skrivas ut "Hej, världen!" med ett mellanslag mellan "Hej," och "världen!".

python
print("Hej", "världen!")
1

1.3 f-strängar ​

F-strängar (f-string) är tillgängliga från och med Python 3.6 och ger ett kraftfullt och enkelt sätt att formatera strängar.

Exempel

python
name = "Alice"
age = 17
print(f"Mitt namn är {name} och jag är {age} år gammal.")
1
2

1.4 str.format() ​

Ett annat sätt att formatera strängar är att använda metoden str.format().

Exempel

python
name = "Alice"
age = 17
print("Mitt namn är {} och jag är {} år gammal.".format(name, age))

1.5 Specialtecken ​

Du kan använda speciella tecken som '\n' för ny rad och '\t' för tab.

Exempel

python
print("Första raden\nAndra raden")
print("Kolumn1\tKolumn2")

2. Inmatning - input() ​

2.1 Grundläggande användning ​

'input()'-funktionen används för att läsa indata från användaren. Den returnerar alltid en sträng (string).

I detta exempel kommer programmet att fråga användaren om deras namn och sedan skriva ut en hälsning.

python
namn = input("Vad heter du? ")
print(f"Hej, {namn}!")
1
2

2.2 Konvertering av indata ​

Eftersom input() alltid returnerar en sträng kan det vara nödvändigt att konvertera denna sträng till andra datatyper.

Exempel

python
age = input("Hur gammal är du? ")
age = int(age)  # Konverterar strängen till ett heltal

#age = int(input("Hur gammal är du? "))
print(f"Du är {age} år gammal.")

Exempel

python
pris = input("Ange priset: ")
pris = float(pris)

#pris = float(input("Ange priset: "))
print(f"Priset är {pris:.2f} kronor.")

2.3 Hantering av felaktig indata ​

För att hantera felaktig indata kan du använda try-except-blocket.

Exempel

python
while True:
    try:
        age = int(input("\nHur gammal är du? "))
        break
    except ValueError:
        print("Ange en giltig siffra för ålder!")

print(f"Du är {age} år gammal.")

Slutsats ​

  • print() används för att skriva ut text och andra objekt till konsolen.
  • input() används för att läsa indata från användaren och returnerar en sträng.
  • Du kan formatera utskrifter med f-strängar eller str.format().
  • Kom ihåg att konvertera indata från input() om det behövs, och hantera eventuella fel med try-except.

Dessa två funktioner är grundläggande men kraftfulla verktyg i Python-programmering och är nödvändiga för interaktivt arbete med användare.
