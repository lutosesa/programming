Sets ​

Python sets är en kraftfull datatyp som används för att lagra unika element i en osorterad datasamling. Till skillnad från listor och tuples, tillåter sets inte dubbletter, vilket gör dem idealiska för att hantera unika data och utföra mängdoperationer som union, intersection och difference.

1. Grundläggande egenskaper ​

  1. Unika Element: Sets innehåller endast unika element. Om du försöker lägga till en dubblett ignoreras den.
  2. Osorterade: Elements i ett set har ingen specifik ordning, vilket innebär att deras position inte är fixerad och inte kan nås med indexering.
  3. Mutabla: Du kan ändra ett set genom att lägga till eller ta bort element.

2. Skapa sets ​

Du kan skapa ett set genom att använda den inbyggda set()-funktionen eller genom att använda måsvingar {}-syntaxen.

Exempel

python
# Skapa ett tomt set
my_empty_set = set()
print(my_empty_set)

# Skapa ett set med 'set()'-funktionen
# notera de dubbla runda parenteserna
my_set = set(("äpple", "banan", "päron")) 
print(my_set)

# Skapa ett set med några element
my_set = {1, 2, 3, 4, 5}
print(my_set)

# Skapa ett set från en lista
my_list = [1, 2, 3, 4, 5, 5, 6]
my_set_from_list = set(my_list)  # {1, 2, 3, 4, 5, 6}
print(my_set_from_list)

3. Dubletter ej tillåtna ​

Sets kan inte ha två objekt med samma värde vilket innebär att sets tar bort dubbleterna.

Exempel

python
my_set = {"äpple", "banan", "päron", "äpple"}
print(my_set)
1
2

Obs:

  • Värdena True och 1 anses vara samma värde i sets och behandlas som dubbletter.
  • Värdena False och 0 anses vara samma värde i sets och behandlas som dubbletter:

Exempel

python
# True och 1 anses vara samma värde:
my_set = {"äpple", "banan", "päron", True, 1, 2}
print(my_set)

# False och 0 anses vara samma värde:
my_set = {"äpple", "banan", "päron", False, True, 0}
print(my_set)

6. Åtkomst till set's element ​

Du kan inte komma åt en element i en set genom att hänvisa till ett index eller en nyckel. Men du kan gå igenom en set's element med en for loop, eller fråga om ett angivet värde finns i en set, genom att använda nyckelordet in.

Exempel

python
my_set = {"äpple", "banan", "päron", "apelsin"}

# FOR-LOOP
for x in my_set:
  print(x, end= " ")  # Output: äpple päron banan apelsin

# Hitta elementet 'päron' 
for element in my_set:
    if element == 'päron':
        print(element)      # Output: päron
        break  # Avsluta loopen när vi har hittat vårt element

# KONTROLLERA OM "banan" finns i 'my_set':
print("banan" in my_set)   # Output:  True

# KONTROLLERA om "banan" INTE finns i 'my_set':
print("banan" not in my_set)  # Output:  False

7. Lägga till, ta bort element ​

Sets har flera inbyggda metoder för att manipulera deras innehåll:

Exempel

python
my_set = {2, 3, 4, 5}
print(my_set)

# LÄGGA TILL ett element
my_set.add(6)
print(my_set)

# TA BORT ett element (kastar KeyError om elementet inte finns)
my_set.remove(3)
print(my_set)

# TA BORT  ett element (kastar inte fel om elementet inte finns)
my_set.discard(3)
print(my_set)

# TA BORT och RETURNERA ett godtyckligt element
removed_element = my_set.pop()
print(removed_element)

# TÖM setet
my_set.clear()
print(my_set)

8. Lägga till annan 'set' eller 'lista' ​

För att lägga till element från en annan 'set' eller 'list' till den aktuella 'set', använd metoden update().

Exempel

python
fruits = {"äpple", "banan", "päron", "apelsin"}
tropical_fruits = {"pineapple", "mango", "papaya"}

# LÄGG TILL element från en annan 'set' 'tropical_fruits' till 'fruits':
fruits.update(tropical_fruits)
print(fruits)

# LÄGG TILL element i en lista till 'fruits'
mylist = ["kiwi", "orange"]
fruits.update(mylist)
print(fruits)

9. Join-Set ​

Det finns flera sätt att ansluta två eller flera sets i Python.

9.1  union() ​

Metoden union() förenar alla objekt från båda sets, returnerar en ny set med alla element från båda sets.

Exempel

python
set_a = {1, 2, 3, 4}
set_b = {3, 4, 5, 6}

union_set = set_a.union(set_b)  

# Alternativ syntax
union_set = set_a | set_b
print(union_set)    # {1, 2, 3, 4, 5, 6}

Slutsats ​

Python sets är en användbar datatyp för att hantera datasamlingar av unika element. De är osorterade, vilket betyder att elementen inte har en fast ordning, och de tillåter inte dubbletter.

Sets är bra för att snabbt kontrollera om ett element finns i en samling och för att utföra olika mängdoperationer som union (kombinera sets), intersection (hitta gemensamma element), och difference (hitta skillnader mellan sets).

Sets är enkla att använda och kraftfulla när du vill arbeta med unika data eller behöver effektivt utföra operationer på stora samlingar av objekt. Genom att förstå och använda sets kan du skriva mer effektiv och tydlig kod i dina Python-projekt.
