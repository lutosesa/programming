Python variabler ​

Inledning ​

I Python används variabler för att lagra och hantera data under programmets körning, som en behållare där du lagrar information inuti ditt program.

En variabel kan ses som en etikett eller ett namn som pekar på ett specifikt värde i minnet. Till skillnad från vissa andra programmeringsspråk kräver Python inte att du deklarerar en variabels typ innan du tilldelar ett värde till den.

Variabler i Python skapas genom tilldelning med likhetstecknet (=), och de kan ändra typ dynamiskt baserat på det tilldelade värdet. Det innebär att du kan börja med att lagra ett heltal i en variabel och senare tilldela en sträng till samma variabel. Detta gör Python flexibelt och lätt att använda, särskilt för nybörjare.

Skapa en variabel ​

Du skapar en variabel genom att tilldela ett värde till den med hjälp av '=' operatorn.

Exempel

python
x = 7
y = 7.5
greeting = "Hello, World!"

print(x)
print(y)
print(greeting)

Exempel

python
x = 16         # x är av typen int (integer-heltal)
x = "Johnny"   # x är nu av typen str (string-text)
print(x)

Variablers utmatning (output) ​

print() ​

Python-funktionen print() används för att skriva ut text och andra data till vanligtvis konsolen eller terminalen. Den är en av de mest grundläggande och användbara funktionerna i Python för att visa information, felsöka kod och kommunicera med användaren.

Med print() funktionen kan man mata ut flera variabler, separerade med ett kommatecken:

Exempel

python
x = "Python"
y = "är"
z = "grymt"
print(x, y, z)  # Output: Python är grymt
1
2
3
4

+ operatorn ​

Du kan också använda + operatorn för att mata ut flera variabler:

Exempel

python
x = "Python "
y = "är "
z = "fantastiskt"
print(x + y + z)   # Output: Python är fantastiskt
1
2
3
4

Obs!

Lägg märke till mellanslagstecken efter "Python " och "är " utan dem skulle resultatet bli "Pythonärfantastiskt".

För siffror + fungerar tecknet som en matematisk operator:

Exempel

python
x = 10
y = 6
print(x + y)   # Output: 16

Med print() funktionen, när man försöker kombinera en sträng och ett nummer med + operatorn, kommer Python att ge ett fel:

Exempel

python

y = 16

Det bästa sättet att mata ut flera variabler med print() funktionen är att separera dem med kommatecken, som till och med stöder olika datatyper:

Exempel

python

y = 16

f-strängar (f-string) ​

F-strängar, eller formaterade strängar, är en funktion i Python som introducerades i version 3.6. De gör det enkelt och läsbart att inkludera uttryck och variabler i strängar. F-strängar skapas genom att placera ett f eller F före strängens början och genom att använda klammerparentes (måsvingar) {} för att inkludera variabler och uttryck direkt i strängen.
F-strängar är det mest användbara sättet att skriva ut text just nu.

Exempel - kombination av text och variabler i en utskrift:

python
x = "Johnny"
y = 16
print(f"{x} är {y} år!")   # Output: Johnny är 16 år!

Regler för variabelnamn (namnkonventioner) ​

Varje variabel måste ha ett namn som är unikt i sitt programsblock. Namnet väljer du själv.
De allmänna reglerna för att namnge variabler är:

  • Variabelnamn måste börja med en bokstav (a-z, A-Z) eller ett understreck (_).
  • Variabelnamn kan innehålla bokstäver, siffror och understreck (a-z, A-Z, 0-9, _).
  • Variabelnamn är skiftlägeskänsliga (dvs. 'myVar' och 'myvar' är olika variabler).
  • Använd lämpliga och beskrivande namn för att göra koden mer läsbar.
  • Ett variabelnamn kan inte vara något av Python-nyckelorden.

Variabelnamn bestående av flera ord ​

Variabelnamn med mer än ett ord kan vara svåra att läsa.
Det finns flera tekniker du kan använda för att göra dem mer läsbara:

Kamel case (fall) ​

Exempel: varje ord, utom det första, börjar med en stor bokstav.

python
myVariableName = "Johnny"
1

Pascal case ​

Exempel: varje ord börjar med en stor bokstav.

python
MyVariableName = "Johnny"
1

Snake case ​

Exempel: varje ord separeras av ett understreck.

python
my_variable_name = "Johnny"
1

Typkonvertering ​

Ibland behöver du konvertera mellan olika datatyper, vilket kan göras med hjälp av inbyggda funktioner som int(), float(), str(), etc.

Exempel

python

y = int(x)    # Konverterar string till heltal
z = float(x)  # Konverterar sträng till decimalttal
print(y, z)

Konstanter ​

Python har ingen inbyggd mekanism för att deklarera konstanter. Konventionellt används variabelnamn med stora bokstäver för att indikera att variabeln är en konstant och inte bör ändras.

Exempel

python
PI = 3.14159
GRAVITY = 9.8

print(PI)
print(GRAVITY)

'None' värde ​

'None' representerar inget värde eller "ingenting". Det används för att visa att en variabel inte har något specifikt värde ännu, eller att en funktion inte returnerar något. Tänk på det som en tom plats eller ett tomt svar.

Exempel

python
x = None
if x is None:
    print("x har inget värde!")

Globala och lokala variabler ​

Variabler som skapas utanför en funktion (som i alla föregående exempel) kallas globala variabler.
Globala variabler kan användas av alla, både inuti funktioner och utanför.

  • Lokala variabler: Deklareras inom en funktion och kan endast användas inom den funktionen.
  • Globala variabler: Deklareras utanför alla funktioner och kan användas både inom och utanför funktioner.

Exempel på lokala variabler

python
def my_function():
    local_var = "Jag är local"
    print(local_var)

my_function()

# Detta skulle ge ett fel eftersom variabeln local_var är lokal för my_function
# print(local_var)

Packa upp en samling (lista, tupple, dictionary) ​

Om du har en samling värden i en lista, tuple etc. Python låter dig extrahera värdena till variabler. Detta kallas uppackning (unpacking)

Exempel: Packa upp en lista:

python
fruits = ["äpple", "banan", "apelsin"]
x, y, z = fruits
print(f"x = {x}")
print(f"y = {y}")
print(f"z = {z}")

Exempel: Packa upp en tupple:

python
fruits = ("äpple", "banan", "apelsin")
(x, y, z) = fruits
print(x)
print(y)
print(z)

my_dict = {'name': 'Johnny', 'age': 16, 'city': 'Västerås'}

Slutsats ​

  • Python-variabler är namn som används för att referera till värden i programmet.
  • Variabler kan lagra olika datatyper såsom heltal (int), decimalttal (float), strängar (str) och listor (list).
  • Tilldelning av värden till variabler sker med likhetstecknet (=).
  • Variabler i Python är dynamiskt typade, vilket innebär att deras typ bestäms vid körning och kan ändras.
  • Namn på variabler måste börja med en bokstav eller ett understreck (_), följt av bokstäver, siffror eller understreck.
  • Att använda beskrivande variabelnamn är god praxis för att göra koden lättare att förstå och underhålla.
