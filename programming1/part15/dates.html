Datum och tid ​

Välkommen till denna genomgång om att använda datum och tid i Python!

En viktig aspekt av många applikationer är hantering av datum och tid. I denna lektion kommer vi att utforska hur man arbetar med datum i Python, från grundläggande operationer till mer avancerade användningsområden.

1. Vad är datum och tid? ​

Datum och tid är sätt att representera specifika punkter i tiden. Datum representerar en specifik dag, månad och år, medan tid representerar en specifik timme, minut och sekund på en dag.

2. Grundläggande datum och tid i Python ​

Python har flera inbyggda moduler för att hantera datum och tid. De mest använda modulerna är time och datetime.

Datatyper för datum och tid ​

De viktigaste datatyperna för datum och tid i Python inkluderar:

  • datetime.date: Representerar ett kalenderdatum (år, månad, dag).
  • datetime.time: Representerar en tid (timme, minut, sekund, mikrosekund).
  • datetime.datetime: Kombinerar datum och tid.
  • datetime.timedelta: Representerar skillnaden mellan två datum eller tider.

Skapa och manipulera datum och tid ​

Här är några grundläggande exempel på hur man skapar och manipulerar datum och tid i Python.

Exempel

python
import datetime

# Skapa ett datumobjekt
datum = datetime.date(2024, 8, 6)
print("Datum:", datum)

# Skapa ett tidsobjekt
tid = datetime.time(14, 30, 0)
print("Tid:", tid)

# Skapa ett datetime-objekt
datum_tid = datetime.datetime(2024, 8, 6, 14, 30, 0)
print("Datum och tid:", datum_tid)

# Manipulera datum och tid
nytt_datum = datum + datetime.timedelta(days=5)
print("Nytt datum:", nytt_datum)

datetime.datetime ​

datetime.datetime kombinerar datum och tid.

Exempel

python
from datetime import datetime
import pytz

# Skapa ett datetime-objekt
local_tz = pytz.timezone('Europe/Stockholm')  # hanterar tidszoner

datetime.timedelta ​

datetime.timedelta används för att representera skillnaden mellan två datum eller tider.

Exempel

python
from datetime import timedelta, datetime
import pytz

# Skapa ett timedelta-objekt
delta = timedelta(days=5, hours=3, minutes=30)
print("Timedelta:", delta)

# Lägg till timedelta till ett datum
local_tz = pytz.timezone('Europe/Stockholm')  # hanterar tidszoner
current_datetime = datetime.now(local_tz)

new_datetime = current_datetime + delta
print("Ny datum och tid:", new_datetime)

4. Formatering av datum och tid ​

Formatering innebär att omvandla datum och tid till strängar eller omvänt, vilket gör dem mer läsbara eller användbara i olika kontexter.

  • Objektet datetime har en metod för att formatera datum-objekt
  • Metoden kallas strftime(), och tar en parameter, format, för att ange formatet för den returnerade strängen:

Exempel

python
from datetime import datetime
import pytz, locale

# Ställ in lokaliseringsinställningarna till svenska
locale.setlocale(locale.LC_TIME, 'sv_SE.UTF-8')

# Skapa ett datetime-objekt
local_tz = pytz.timezone('Europe/Stockholm')  # hanterar tidszoner
current_datetime = datetime.now(local_tz)

# Formatera till sträng
formatted_date = current_datetime.strftime("%Y-%m-%d %H:%M:%S")
print("Formaterad datum och tid:", formatted_date)
print("Dag:", current_datetime.strftime("%A"))
print("Månad:", current_datetime.strftime("%B"))

# Omvandla sträng till datetime
parsed_date = datetime.strptime("2024-08-06 14:30:00", "%Y-%m-%d %H:%M:%S")
print("\nParsed datum och tid:", parsed_date)

5. Tidszoner och pytz ​

För att hantera tidszoner i Python används pytz-modulen.

Exempel

python
import pytz

# Hämta alla tidszoner för Europa med en List Comprehensions
european_timezones = [tz for tz in pytz.all_timezones if tz.startswith('Europe/')]

# Skriv ut de europeiska tidszonerna
for tz in european_timezones:
    print(tz)

from datetime import datetime

sweden_datetime = current_datetime.strftime("%H:%M:%S")  # %Y-%m-%d %H:%M:%S

# Konvertera till en annan tidszon, och sedan
# formaterar till en sträng med tiden i New York
new_york_tz = pytz.timezone('America/New_York')
new_york_time = current_datetime.astimezone(new_york_tz).strftime("%H:%M:%S")
print("Tid i New York:", new_york_time )

6. Praktiska exempel ​

Exempel 1 ​

Ex. Beräkna ålder

python
from datetime import date

def calculate_age(birthdate):
    today = date.today()
    age = today.year - birthdate.year - ((today.month, today.day) < (birthdate.month, birthdate.day))
    return age

birthdate = date(2008, 8, 8)
print("Ålder:", calculate_age(birthdate))

local_tz = pytz.timezone('Europe/Stockholm')

#

time_remaining = event_date - current_datetime
