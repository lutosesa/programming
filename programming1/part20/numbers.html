Python nummer (tal) ​

Python erbjuder ett brett spektrum av numeriska datatyper för att hantera olika typer av tal.
'int' och 'float' är de vanligaste typerna för heltal och decimaltal, medan 'complex' används för komplexa tal.

1. Grundläggande numeriska datatyper ​

Python har flera inbyggda numeriska datatyper:

  • int: Heltal, t.ex. 3, 42, -7
  • float: decimaltal, t.ex. 3.14, -0.001
  • complex: Komplexa tal, t.ex. 1+2j, 3-4j
python
# Heltal
a = 5
b = -3

# decimaltal
c = 3.14
d = -0.001

# Komplexa tal
e = 2 + 3j
f = -1 - 1.5j

print(f"heltal: {a}, {b}")
print(f"decimaltal: {c}, {d}")
print(f"komplex tal: {e} och {f}")

2. Aritmetiska operationer ​

Python stödjer grundläggande aritmetiska operationer som addition, subtraktion, multiplikation och division samt mer avancerade operationer som potentiation och moduloräkning.

python
x = 10
y = 3

# Addition
add = x + y  # 13

# Subtraktion
sub = x - y  # 7

# Multiplikation
mul = x * y  # 30

# Division
div = x / y  # 3.333...

# Heltalsdivision
floordiv = x // y  # 3

# Modulo (resten av divisionen)
mod = x % y  # 1

# Potentiation
exp = x ** y  # 1000

print(f"Addition: {add} \nSubtraktion: {sub} \nMultiplikation: {mul}")
print(f"Division: {div:.2f} \nHeltalsdivision: {floordiv}")
print(f"Modulo (resten av divisionen): {mod} \nPotens: {exp}")

3. Inbyggda numeriska funktioner ​

Python erbjuder många inbyggda funktioner för numeriska operationer. Här är några av de mest användbara:

python
# Absolutvärde
abs_value = abs(-7)  # 7

# Runda ett flyttal
rounded_value = round(3.14159, 2)  # 3.14

# Maximalt och minimalt värde
max_value = max(1, 5, 3)  # 5
min_value = min(1, 5, 3)  # 1

# Summan av en lista
sum_value = sum([1, 2, 3, 4])  # 10

print(f"Absolutvärde: {abs_value} \nRunda ett flyttal: {rounded_value}")
print(f"Maximalt och minimalt värde: {max_value} och {min_value}")
print(f"Summan av en lista: {sum_value}")

4. Random tal ​

Python erbjuder random-modulen för att generera slumpmässiga tal.

python
import random

# Slumpmässigt heltal mellan två värden
rand_int = random.randint(1, 10)  # 1 <= rand_int <= 10
print(f"Slumpmässigt heltal mellan 1 och 10: {rand_int}")

# Slumpmässigt decimaltal mellan 0 och 1
rand_float = random.random()
print(f"Slumpmässigt decimaltal mellan 0 och 1: {rand_float:.2f}")

# Slumpmässigt val från en lista
rand_choice = random.choice([1, 2, 3, 4, 5])
print(f"Slumpmässigt val från en lista: {rand_choice}")

5. Användbara bibliotek ​

Python har flera kraftfulla bibliotek för numerisk beräkning, varav de mest populära är math och numpy.

5.1 Math-biblioteket ​

Math-biblioteket innehåller många funktioner för matematiska operationer.

python
import math

# Kvadratroten av ett tal
x = 16
sqrt_value = math.sqrt(x)  # 4.0
print(f"Kvadratroten av {x}: {sqrt_value}")

# Naturliga logaritmen
y = 2.71828
log_value = math.log(y)  # ≈ 1.0
print(f"Naturliga logaritmen av {y}: {log_value:.7f}")

# Sinus av en vinkel (i radianer)
z = math.pi / 2
sin_value = math.sin(z)  # 1.0
print(f"Sinus av {z:.3f} (i radianer): {sin_value}")

5.2 Numpy-biblioteket ​

Numpy är ett bibliotek för vetenskapliga beräkningar som stöder stora, flerdimensionella arrays och matriser, samt en samling matematiska funktioner för att arbeta med dessa arrays.

python
import numpy as np

# Skapa arrays
array1 = np.array([1, 2, 3, 4])
array2 = np.array([4, 5, 6, 7])

# Aritmetiska operationer på arrays
sum_arrays = array1 + array2  # Elementvis addition
product_arrays = array1 * array2  # Elementvis multiplikation
print(f"\nsum_arrays: {sum_arrays}\nproduct_arrays: {product_arrays}")

# Matrisoperationer:
matrix_product = np.dot(array2, array2)  # Matrisprodukt
print(f"matrix_product: {matrix_product}")

# Statistikfunktioner
mean_value = np.mean(array1)  # Medelvärde
std_value = np.std(array1)  # Standardavvikelse
print(f"mean_value: {mean_value} \nstd_value: {std_value:.3f}\n")

5.3 Installera numpy på VS Code ​

  • öppna VS Code
  • öppna mappen där du har dina Python script (övningar, uppgifter), prog1.
  • öppna en Terminal på menyfliken Terminal och klicka på New Terminal.
  • Skapa en virtuell miljö för kursens övningar och uppgifter genom att göra följande på Terminalen:

    • skriv på terminalen följande: python -m venv myenv

    • tryck ner på tangenten Enter,

    • det skapas en mapp som heter myenv

    • det dyker upp ett dialog-fönstret med ett meddemande att ett mytt miljö har skapats,

    • klicka på Yes i dialog-fönstret,

    • Aktivera ditt virtuell miljö (myenv) genom att skriva följande på terminalen: myenv\Scripts\activate

    • tryck ner på tangenten Enter,

    • Nu har du aktivera ditt virtuell miljö (virtual environment):

    • Sedan intallerar du numpy genom att skriva på terminalen: pip install numpy

    • tryck ner på tangenten Enter.

    • klart!

6. Hantering av komplexa tal ​

Komplexa tal i Python hanteras med complex-datatypen och har inbyggda attribut och metoder för att extrahera den reella och imaginära delen.

python
z = 2 + 3j
print(f"z = {z}")

# Reell del
real_part = z.real  # 2.0
print(f"Reell del: {real_part}")

# Imaginär del
imaginary_part = z.imag  # 3.0
print(f"Imaginär del: {imaginary_part}")

# Absolutvärde (modul)
abs_value = abs(z)  # 3.605551275463989
print(f"Absolutvärde (modul): {abs_value}")

# Konjugat
conjugate_value = z.conjugate()  # 2 - 3j
print(f"Konjugat: {conjugate_value}")

7. Felsökning och undantag ​

Vid numeriska operationer kan det uppstå fel. Det är viktigt att kunna hantera undantag för att undvika att program kraschar.

python
try:
    result = 10 / 0
    print (f"Kvoten: {result}")
except ZeroDivisionError:
    print("Kan inte dividera med noll!")

Slutsats ​

  • Python hanterar nummer med flera inbyggda datatyper, främst heltal (int), decimaltal (float) och komplexa tal (complex).
  • Heltal (int) används för att representera exakta siffror utan decimaler,
  • medan decimaltal (float) används för att representera siffror med decimaler.
  • Python stöder också komplexa tal (complex), som består av en realdel och en imaginärdel.
  • Dessa datatyper möjliggör en mängd matematiska operationer och funktioner, som tillhandahålls av både inbyggda operatorer och bibliotek som math och numpy.
  • Python hanterar automatiskt konverteringar mellan dessa typer vid behov, vilket gör det enkelt att arbeta med tal i olika former och storlekar.
  • Sammanfattningsvis erbjuder Python ett flexibelt och kraftfullt system för att arbeta med olika typer av numeriska data.
