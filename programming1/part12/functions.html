Python funktioner ​

Python är ett kraftfullt och lättanvänt programmeringsspråk som används över hela världen för allt från webbapplikationer till datavetenskap. En av de mest grundläggande och viktiga koncepten inom Python (och andra programmeringsspråk) är funktionen.

Denna del kommer att ge en omfattande genomgång av funktioner i Python, från grundläggande definitioner till mer avancerade användningsområden.

1. Vad är en funktion? ​

En funktion är en block av kod som kan återanvändas flera gånger i ett program. Det är en självständig del av programmet som utför en specifik uppgift.

2. Definiera en funktion ​

För att definiera en funktion i Python använder man def nyckelordet följt av funktionsnamnet och parenteser. Inuti parenteserna kan man skriva eventuella parametrar som funktionen tar emot.

python
def welcome():
    print("Hej, Gymnasieelever!")

4. Parameterar och argument ​

Parametrar är variabler som anges inom parenteserna när du definierar en funktion. Argument är de värden du skickar till funktionen när du anropar den.

python
def welcome(name):
    print(f"Hej, {name}!")

welcome("Kalle Anka")

6. Returnera flera värde ​

En funktion kan returnera flera värden som en tuple.

python
def operera(x, y):
    summa = x + y
    skillnad = x - y
    produkt = x * y
    kvot = x / y
    return summa, skillnad, produkt, kvot

resultat = operera(10, 2)
print(resultat)

9. *args, godtyckliga argument ​

Om du inte vet i förväg hur många argument som kommer skickas kan du lägga till en asterisk * före parametern args eller ett annat parameternamn.

På så sätt kommer funktionen att ta emot en tuple av argument och kan komma att argumenten.

python
def godtyckliga_argument(*args):
    for arg in args:
        print(arg)

def annan_funktion(*barn):
  print("Det yngsta barnet är " + barn[2])

# Anropar funktionerna
godtyckliga_argument(1, 2, 3)
annan_funktion("Emil", "Tobias", "Linus")

10. **kwargs, godtyckliga argument ​

Om du inte vet hur många nyckelordargument som kommer att skickas till din funktion, lägg till två asterisker ** före parameternamnet i funktionsdefinitionen.

På detta sätt kommer funktionen att ta emot en dictionary med argument och kan komma åt objekten.

python
def godtyckliga_nyckelordsargument(**kwargs):
    for nyckel, värde in kwargs.items():
        print(f"{nyckel}: {värde}")
    # eller
    print(f"\nHon heter {kwargs['namn']}!")

godtyckliga_nyckelordsargument(namn="Anna", ålder=16)

13. Rekursiva funktioner ​

Rekursion är ett vanligt matematiskt och programmeringsbegrepp. Det betyder att en funktion anropar sig själv.

Exempel

python
def factorial(n):
    if n == 1:
        return 1
    else:
        return n * factorial(n - 1)    # n! = n * (n - 1)!

print(factorial(5))

Förklaring

  • En rekursiv funktion anropar sig själv för att lösa ett problem.
  • Den rekursiva funktionen factorial(5) bygger på definitionen av fakultet som är en funktion inom matematiken.
5! = 5 * 4 * 3 * 2 * 1 = 120

Avslut ​

Funktioner är en grundläggande byggsten i Python-programmering som låter dig skriva återanvändbar och modulär kod. Genom att förstå hur man definierar, anropar, och använder funktioner kan du skapa mer effektiva och läsbara program.
Fortsätt öva och experimentera med olika typer av funktioner för att förbättra dina programmeringskunskaper.
