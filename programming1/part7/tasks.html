Uppgifter om Python Villkorsatser if-elif-else ​

Uppgifterna nedan bygger på avsnittet om Del 7 - Python Villkorsatser if-elif-else.

Uppgifterna är obligatoriska och dessa kommer att hjälpa dig att förstärka din förståelse på "Villkorsatser if-elif-else" genom praktiska tillämpningar.

Uppgift 1: Grundläggande If-sats Syntax ​

Skriv ett program som frågar användaren efter deras ålder och sedan skriver ut ett meddelande baserat på om de är över 18 eller inte.

Instruktioner:

  • Använd en if-sats för att kontrollera om ålder är större än eller lika med 18 .
  • Om användaren är över 18, skriv ut "Du är myndig".
  • Annars, skriv ut "Du är inte myndig ännu".

Exempel på indata:

python
Skriv in din ålder: 17

Exempel på utdata:

python
Du är inte myndig ännu.

Uppgift 2: Använda Elif och Else ​

Skriv ett program som bedömer en elevs betyg baserat på deras poäng.

Instruktioner:

  1. Be användaren ange ett poängtal mellan 0 och 100.
  2. Använd if-elif-else-satser för att tilldela ett betyg:
    • Poäng >= 90: "A"
    • Poäng >= 80: "B"
    • Poäng >= 70: "C"
    • Poäng >= 60: "D"
    • Poäng >= 50: "E"
    • Poäng < 50: "F"

Exempel på indata:

python
Ange dina poäng: 85

Exempel på utdata:

python
Ditt betyg är: B

Uppgift 3: Inbäddade If-satser ​

Skriv ett program som tar två tal från användaren och kontrollerar vilket av dem som är störst. Om de är lika stora, skriv att de är lika.

Instruktioner:

  1. Be användaren skriva in två tal.
  2. Använd en if-sats för att kontrollera om det första talet är större än det andra.
  3. Använd en inbäddad if-sats för att kontrollera om de är lika stora.
  4. Skriv ut ett lämpligt meddelande i varje fall.

Exempel på indata:

python
Ange första talet: 10
Ange andra talet: 10

Exempel på utdata:

python
Talen är lika.

Uppgift 4: Sammansatta Villkor med Logiska Operatorer ​

Skriv ett program som tar en temperatur i Celsius och kontrollerar om det är en varm, kall eller normal temperatur.

Instruktioner:

  1. Be användaren ange en temperatur i Celsius.
  2. Använd logiska operatorer (and, or) för att avgöra:
    • Om temperaturen är över 30, skriv ut "Det är varmt".
    • Om temperaturen är mellan 15 och 30, skriv ut "Temperaturen är normal".
    • Om temperaturen är under 15, skriv ut "Det är kallt".

Exempel på indata:

python
Ange temperaturen i Celsius: 10

Exempel på utdata:

python
Det är kallt.

Uppgift 5: Villkor med flera satser ​

Skriv ett program som ber användaren om ett tal och kontrollerar om talet är positivt, negativt eller noll. Dessutom ska det kontrollera om talet är jämnt eller udda.

Instruktioner:

  1. Be användaren ange ett tal.
  2. Kontrollera först om talet är:
    • positivt,
    • negativt eller
    • noll.
  3. Därefter, kontrollera om talet är jämnt eller udda.
  4. Skriv ut ett meddelande som kombinerar båda resultaten.

Exempel på indata:

python
Ange ett tal: 4

Exempel på utdata:

python
Talet är positivt och jämnt.
