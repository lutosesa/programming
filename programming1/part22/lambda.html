Python Lambda funktion ​

Python är ett populärt programmeringsspråk känt för sin enkelhet och läsbarhet. Bland dess många kraftfulla funktioner finns lambda-funktioner, även kallade 'anonyma funktioner'. Dessa är små, korta funktioner som kan definieras utan ett namn.

Denna genomgång är utformad för att ge dig en grundläggande förståelse av lambda-funktioner, deras syfte och hur man använder dem.

1. Vad är lambda-funktion? ​

En lambda-funktion i Python är en liten, anonym funktion som definieras med hjälp av lambda nyckelordet. Till skillnad från vanliga funktioner som definieras med def nyckelordet, är *lambda-funktioner begränsade till en enda rad uttryck.

2. Definition av lambda ​

En lambda-funktion definieras med lambda följt av indata (argument), ett kolon och sedan uttrycket som returnerar ett värde. Syntaxen är:

python
lambda argument: uttryck
1

Här är ett enkelt exempel:

python
dubbel = lambda x: x * 2
print(dubbel(5))  # Output: 10
1
2

3. Jämförelse mellan 'lambda' och vanliga funktioner ​

Här är en jämförelse mellan en vanlig funktion och en lambda-funktion som utför samma uppgift.

Vanlig funktion:

python
def kvadrat(x):
    return x * x

print(kvadrat(4))  # Output: 16
1
2
3
4

Lambda-funktion:

python
kvadrat = lambda x: x * x
print(kvadrat(4))  # Output: 16
1
2

4. Varför lambda? ​

Kraften i lambda visas bättre när du använder dem som en anonym funktion i en annan funktion.

Säg att du har en funktionsdefinition som tar ett argument, och det argumentet kommer att multipliceras med ett okänt tal:

Exempel på att skapa en funktion som dubblar antalet du skickar in, eller tredubblar numret du skuckar in:

python
def myfunc(n):
  return lambda a : a * n

mydoubler = myfunc(2)
mytripler = myfunc(3)

tal = 11
print(f"Dubblar talet {tal}: {mydoubler(tal)}")
print(f"Tredubblar talet {tal}: {mytripler(tal)}")
1
2
3
4
5
6
7
8
9

Obs:

Använd lambdafunktioner när en anonym funktion krävs under en kort tidsperiod.

Används ofta som argument till andra funktioner eller i kombination med inbyggda funktioner som map(), filter(), och reduce().

5. Lambda med map() ​

map() är en inbyggd funktion som tillämpar en given funktion på varje element i en iterable (t.ex. en lista) och returnerar en map-objekt (som kan konverteras till en lista).

Exempel

python
tal = [1, 2, 3, 4, 5]  # en lista

dubblade_tal = map(lambda x: x * 2, tal)

print(list(dubblade_tal))  # Output: [2, 4, 6, 8, 10]
1
2
3
4
5

6. Lambda med filter() ​

*filter() är en inbyggd funktion som filtrerar element i en iterable baserat på en funktion som returnerar sant eller falskt.

Exempel

python
tal = [1, 2, 3, 4, 5]  # en lista

udda_tal = filter(lambda x: x % 2 != 0, tal)

print(list(udda_tal))  # Output: [1, 3, 5]
1
2
3
4
5

7. Lambda med reduce() ​

reduce() är en del av functools modulen och används för att tillämpa en funktion kumulativt på elementen i en iterable, från vänster till höger, så att den reduceras till ett enda värde.

Exempel

python
from functools import reduce

tal = [1, 2, 3, 4, 5]  # en lista

summa = reduce(lambda x, y: x + y, tal)
print(summa)  # Output: 15
1
2
3
4
5
6

Avslut ​

Lambda-funktioner är en kraftfull och flexibel del av Python som gör det möjligt att skriva korta och koncisa funktioner. De är särskilt användbara när du behöver en snabb lösning för enkla operationer, särskilt när de används tillsammans med funktioner som map(), filter(), och reduce().

Genom att förstå och använda lambda-funktioner kan du skriva mer effektiv och läsbar kod. Fortsätt att öva på olika exempel och övningar för att stärka din förståelse och skicklighet med lambda-funktioner i Python.
