Python while-loop ​

I programmering är det ofta nödvändigt att upprepa en block kod tills ett visst villkor är uppfyllt. Detta kallas för iteration och ett av de mest grundläggande sätten att uppnå detta i Python är genom att använda en while-loop.

Denna genomgång kommer att förklara:

  • vad en while-loop är,
  • hur den fungerar och
  • när den ska användas.

Vi kommer att gå igenom flera exempel och förklara varje del av koden noggrant.

1. Vad är en While-loop? ​

En while-loop är ett sätt att få ett program att upprepa en viss kod flera gånger, så länge ett specifikt villkor är uppfyllt (sant). Koden i loopen fortsätter alltså att köras om och om igen tills villkoret blir falskt.

Det är ett viktigt verktyg när vi behöver göra något många gånger utan att skriva samma kod om och om igen.

2. Grundläggande syntax för 'while-loop' ​

Syntaxen för en while-loop i Python ser ut så här:

python
while villkor:
    # kodblock som körs om villkoret är sant
1
2
  • while är nyckelordet som startar loopen.
  • villkor är ett uttryck som utvärderas till sant (True) eller falskt (False).
  • kodblock är indenterad (indrag) och körs upprepade gånger så länge villkoret är sant (True).

Låt oss titta på några exempel för att bättre förstå hur while-loop fungerar.

2.1 Exempel ​

Exempel på en enkel while-loop

python
nummer = 1              # Vi startar med nummer 1

while nummer <= 5:      # villkor: så lönge nummer är mindre än eller lika med 5.
    print(nummer)       # skriv ut det aktuella värdet av variabeln nummer.
    nummer += 1         # lägg till 1 till nummer varje gång loopen körs
1
2
3
4
5

Vad händer här:

  1. Loopen startar och kontrollerar om nummer är mindre än eller lika med 5.
  2. Eftersom det är sant (första gången är nummer 1), körs koden inuti loopen. Programmet skriver ut 1 och ökar nummer med 1.
  3. Loopen körs igen, och nu är nummer 2, vilket fortfarande uppfyller villkoret. Detta fortsätter tills nummer blir 6, då villkoret blir falskt och loopen slutar.

2.2 Exempel ​

Exempel på while-loop med användarinmatning

python
password = "python123"
my_password = ""

  • Variabeln 'my_password' sätts initialt till en tom sträng.
  • Loopen fortsätter att be användaren om att skriva in lösenordet tills det korrekta lösenordet skrivs in.
  • När rätt lösenord skrivs in, avslutas loopen och "Accessen beviljad!" skrivs ut.

3. Oändliga loopar och hur man undviker dem ​

En oändlig loop uppstår när villkoret i en while-loop aldrig blir falskt. Detta kan få ditt program att fastna i en evig cykel och sluta svara.

3.1 Exempel ​

Exempel på en oändlig loop. Detta ska man undvika för att programmet kommer att krascha.

python
while True:
    print("Den här slingan kommer att köra för evigt!")
1
2

För att undvika oändliga loopar, se till att ditt **villkor så småningom blir falskt. Detta kan innebära att du behöver uppdatera en variabel inom loopen eller inkludera en brytmekanism, som t.ex. break.

3.2 Exempel ​

Exempel på att använda break

python
while True: 
    exit = input("Den här slingan kommer att köra för evigt,\nOM du INTE mata in 'x': ")
    if exit == 'x':     
        break

2
3
4

Förklaring:

  • Rad 2: While-Loopen utmanar användaren att mata in 'x' för att avbryta den eviga loopen. 'x'-värdet tilldelas till variabeln 'exit'
  • Rad 3: Det finns en kontroll-if-sats som kollar om användaren har matat in 'x'. Om användaren har gjort det anvbryts den eviga loopen med nyckelordet break.

4. break-sats ​

Nyckelordet break används för att stoppa en loop direkt, både while-loopar' och for-loopar, även om loopen inte har kört klart eller om villkoret fortfarande är sant. Det låter programmet "avbryta" (avsluta) loopen och hoppa till nästa del av koden utanför loopen.

Hur fungerar break? ​

Tänk dig att du har en loop som körs om och om igen. Om du använder break, säger du till programmet att sluta köra loopen och hoppa direkt ut ur loopen, även om loopen normalt skulle ha fortsatt.

4.1 Exempel ​

Exempel - Undvika oändlig loop

python
# Loopen körs för alltid om vi inte stoppar den
while True:
    exit = input("<Skriv 'stopp' för att avsluta> ")
    if exit  == 'stopp':        
        break
    print(f"Du skrev: {exit}")

print("\nProgrammet är avslutat!")

Förklaring:

  • Loopen körs oändligt eftersom villkoret alltid är 'True'.
  • Användaren ombeds skriva något.
  • Om användaren skriver "stopp", körs nyckelordet 'break' och loopen avslutas.
  • Om användaren skriver något annat, fortsätter loopen att köras.

4.2 Exempel ​

Exempel på en tidig avslutning av en loop:

python
nummer = 0

while nummer < 10:
    print(nummer)
    if nummer == 5:

        break
    nummer += 1

  • Om 'nummer' når 5, skrivs ett meddelande ut och 'break' avslutar loopen.
  • Om 'nummer' inte är 5, fortsätter loopen att öka nummer med 1 för varje iteration.

5. continue-sats ​

Nyckelordet continue används i Python för att hoppa över den återstående koden i den aktuella iterationen av en loop och fortsätta med nästa iteration. Det kan användas i både while-loopar' och for-loopar.

Hur fungerar continue? ​

Om du använder continue i en loop, säger du till programmet att hoppa över det som finns kvar av loopen just nu och börja om på nytt med nästa varv/loop. Loopen fortsätter alltså köra som vanligt, men den hoppar över den kod som kommer efter continue just för det varvet/loop.

5.1 Exempel ​

Exempel på att hoppa över jämna nummer

python
nummer = 0
while nummer < 10:
    nummer += 1
    if nummer % 2 == 0:     
        continue
    print(nummer)

  • Loopen körs medan nummer är mindre än 10.
  • nummer ökas med 1 i början av varje iteration/varv.
  • Om nummer är jämnt (nummer % 2 == 0), hoppar 'continue' över resten av koden i den aktuella iterationen/varvet.
  • Endast udda nummer skrivs ut.

  • Om användaren skriver något annat, skrivs inmatningen ut.

6. else-sats ​

I Python kan du använda en else -sats tillsammans med en while-loop. Denna else-sats körs när loopens villkor blir falskt och loopen avslutas normalt. Om loopen avslutas med hjälp av en break-sats', kommer else-sats' inte att köras.

6.1 Exempel ​

Exempel på en enkel användning av else-sats

python
räknare = 0
while räknare < 5:
    print(räknare)
    räknare += 1
else:
    print("\nLoopen avslutades normalt.")

Förklaring:

  • Loopen körs medan 'räknare' är mindre än 5.
  • 'räknare' ökas med 1 för varje iteration/varv.
  • När 'räknare' blir 5, blir villkoret falskt och loopen avslutas.
  • else-satsen körs och skriver ut "Loopen avslutades normalt."

6.2 Exempel ​

Exempel på användning av else-sats med break

python
räknare = 0
while räknare < 5:
    if räknare == 3:
        break
    print(räknare)
    räknare += 1
else:
    print("\nLoopen avslutades normalt.")

6.3 Exempel ​

Exempel på att söka i en lista med else-sats

python
nummer = [1, 2, 3, 4, 5]
sökning = 3
hittad = False
index = 0

while index < len(nummer):
    if nummer[index] == sökning:
        hittad = True
        break
    index += 1
else:
    print(f"{sökning} hittades inte i listan.")

if hittad:
    print(f"{sökning} hittades vid index {index}.")

  • Om loopen avslutas utan att numret hittas (utan break), körs else-satsen och skriver ut att numret inte hittades.
  • Efter loopen kontrolleras hittad-flaggan om det är True (sant) och resultatet skrivs ut.

Avslut ​

En while-loop är ett kraftfullt verktyg i Python som tillåter dig att upprepa en viss kod flera gånger så länge ett villkor är sant.

Genom att förstå och använda while-loop effektivt kan du skapa mer dynamiska och responsiva program.
