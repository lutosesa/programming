Python while-loop ​

I programmering är det ofta nödvändigt att upprepa en sekvens av operationer tills ett visst villkor är uppfyllt. Detta kallas för iteration, och ett av de mest grundläggande sätten att uppnå detta i Python är genom att använda en while-loop.

Denna genomgång syftar till att förklara vad en while-loop är, hur den fungerar och när den ska användas. Vi kommer att gå igenom flera exempel och förklara varje del av koden noggrant.

1. Vad är en While-loop? ​

En 'while-loop' är en kontrollstruktur som låter dig repetera en block av kod så länge ett specifikt villkor är sant. Den kontrollerar villkoret innan varje iteration, vilket innebär att om villkoret aldrig är sant, kommer koden inuti loopen aldrig att köras.

2. Grundläggande syntax för 'while-loop' ​

Syntaxen för en 'while-loop' i Python ser ut så här:

python
while villkor:
    # kodblock som körs om villkoret är sant
1
2
  • while är nyckelordet som startar loopen.
  • villkor är ett uttryck som utvärderas till True eller False.
  • kodblock är indenterad (indrag) och körs upprepade gånger så länge villkoret är True.

3. Exempel på användning av While-loop ​

Låt oss titta på några exempel för att bättre förstå hur 'while-loop' fungerar.

3.1 Exempel 1 ​

Exempel på en enkel while-loop

python
i = 0
while i < 5:  // [!code warning]
    print(i)  // [!code warning]
    i += 1    // [!code warning]

  • Variabeln 'i' sätts till 0.
  • Loopen fortsätter så länge 'i' är mindre än 5.
  • print(i) skriver ut värdet av 'i''.
  • 'i += 1', (motsavarande 'i = i + 1') ökar värdet av 'i' med 1 efter varje iteration.

  • Loopen fortsätter att be användaren om att skriva in lösenordet tills det korrekta lösenordet "python123" skrivs in.
  • När rätt lösenord skrivs in, avslutas loopen och "Accessen beviljad!" skrivs ut.

4. Oändliga loopar och hur man undviker dem ​

En oändlig loop uppstår när villkoret i en 'while-loop' aldrig blir falskt. Detta kan få ditt program att fastna i en evig cykel och sluta svara.

4.1 Exempel 1 ​

Exempel på en oändlig loop

python
while True:
    print("This loop will run forever!")
1
2

För att undvika oändliga loopar, se till att ditt villkor så småningom blir falskt. Detta kan innebära att du behöver uppdatera en variabel inom loopen eller inkludera en brytmekanism, som t.ex. 'break'.

4.2 Exempel 2 ​

Exempel på att använda break

python
i = 0
while i < 10:
    print(i)
    if i == 5:
        break
    i += 1

5. break-sats ​

Nyckelordet 'break' är en viktig kontrollstruktur som används i Python för att avbryta en loop innan den naturligt skulle ha slutat. Det kan användas i både 'while-loopar' och 'for-loopar'. När 'break'-nyckelordet körs, avslutas loopen omedelbart och programkontrollen flyttas till första raden av kod som följer efter loopen.

5.1 Exempel 1 ​

Exempel - Undvika oändlig loop

python
while True:
    user_input = input("Skriv 'stopp' för att avsluta: ")
    if user_input.lower() == 'stopp':
        break
    print(f"Du skrev: {user_input}")

print("\nProgrammet är avslutat!")

  • Om användaren skriver "stopp", körs 'break' och loopen avslutas.
  • Om användaren skriver något annat, fortsätter loopen.

Exempel på en tidig avslutning av en loop:

  • Om 'number' inte är 5, fortsätter loopen att öka number med 1 för varje iteration.

6. continue-sats ​

Nyckelordet 'continue' används i Python för att hoppa över den återstående koden i den aktuella iterationen av en loop och fortsätta med nästa iteration. Det kan användas i både 'while-loopar' och 'for-loopar'. När 'continue'-nyckelordet körs, avbryts den pågående iterationen och loopens villkor kontrolleras för att avgöra om nästa iteration ska köras.

6.1 Exempel 1 ​

Exempel på att hoppa över jämna nummer

python
number = 0
while number < 10:
    number += 1
    if number % 2 == 0:
        continue
    print(number)

  • Loopen körs medan number är mindre än 10.
  • number ökas med 1 i början av varje iteration.
  • Om number är jämnt (number % 2 == 0), hoppar 'continue' över resten av koden i den aktuella iterationen.
  • Endast udda nummer skrivs ut.

Exempel på filtrering av inmatningar

while True:

        break

  • Om användaren skriver något annat, skrivs inmatningen ut.

7. else-sats ​

I Python kan du använda en 'else-sats' tillsammans med en 'while-loop'. Denna 'else-sats' körs när loopens villkor blir falskt och loopen avslutas normalt. Om loopen avslutas med hjälp av en 'break-sats', kommer 'else-sats' inte att köras.

7.1 Exempel 1 ​

Exempel på en enkel användning av else-sats

python
count = 0
while count < 5:
    print(count)
    count += 1
else:
    print("\nLoopen avslutades normalt.")

  • Loopen körs medan 'count' är mindre än 5.
  • 'count' ökas med 1 för varje iteration.
  • När 'count' blir 5, blir villkoret falskt och loopen avslutas.
  • else-satsen körs och skriver ut "Loopen avslutades normalt."

7.2 Exempel 2 ​

Exempel på användning av else-sats med break

python
count = 0
while count < 5:
    if count == 3:
        break
    print(count)
    count += 1
else:
    print("\nLoopen avslutades normalt.")

        found = True
        break
    index += 1

  • Efter loopen kontrolleras found-flaggan och resultatet skrivs ut.

8. Kombination av while-loop med andra loopar och kontrollstrukturer ​

Du kan kombinera 'while-loop' med andra loopar och kontrollstrukturer som 'for-loop' och 'if-else' för mer komplexa logiker.

Exempel ​

Exempel på kapsling av 'while-loop' och 'for-loop'

python
i = 1
while i <= 3:
    print(f"Yttre loop iteration: {i}")
    for j in range(1, 4):
        print(f"  Inre loop iteration: {j}")
    i += 1

  • Den yttre 'while-loopen' körs 3 gånger.
  • För varje iteration av den yttre loopen körs den inre 'for-loopen' 3 gånger.

Avslut ​

En 'while-loop' är ett kraftfullt verktyg i Python som tillåter dig att upprepa operationer baserat på ett villkor. Genom att förstå och använda 'while-loopar' effektivt kan du skapa mer dynamiska och responsiva program. Öva på att använda 'while-loopar' i olika sammanhang för att bygga upp din förståelse och färdighet i programmering.
