Pygames Surfaces ​

Inledning ​

När du skapar spel med Pygame, är ett av de viktigaste begreppen att förstå surfaces (ytor). En surface är en rektangulär yta där du kan rita grafik, text eller bilder, och den är hjärtat i Pygames grafiksystem. Alla visuella element i ett spel, som spelkaraktärer, bakgrunder och objekt, ritas på en surface innan de visas på skärmen.

Vad är en Surface? ​

I Pygame representerar en surface en yta eller ett område där du kan rita saker. Du kan tänka på en surface som ett "tomt papper" där du kan placera olika objekt som bilder, former eller text.

Varje Pygame-fönster har minst en stor surface, som representerar hela skärmen, men du kan också skapa mindre surfaces för att hantera olika delar av spelet (t.ex. karaktärer eller objekt).

Grundläggande koncept och delar ​

Här är några viktiga saker att förstå när det gäller Pygame surfaces:

  1. Skärm-surface (main display surface): Detta är den huvudsakliga surface som visas på skärmen. Det är hit alla andra surfaces till slut "blitas" (kopieras) innan de visas för spelaren.

  2. Skapa en ny surface: Du kan skapa en ny surface med pygame.Surface() om du vill rita eller hålla grafik innan du kopierar den till skärmen.

  3. Blit: Detta är hur du kopierar en surface till en annan, oftast från en mindre surface till skärmens surface.

  4. Uppdatera skärmen: När du har blitat (kopierat) alla grafiska element på skärmen, använder du pygame.display.flip() eller pygame.display.update() för att visa det.

Exempel på hur surfaces används i Pygame ​

Här är ett enkelt exempel på hur surfaces fungerar:

Exempel 1

python
# STEG 1: Importera bibliotek
import pygame   
import sys       # Importera sys för att avsluta programmet

# STEG 2: Initiera Pygame
pygame.init()

# STEG 3: Skapa ett spelfönster
screen = pygame.display.set_mode((800, 600))

# STEG 4: Sätt fönstertitel
pygame.display.set_caption("Exempel med surfaces")

# STEG 5: Sätt fönstrets grundfärg, svart (RGB-färgkod)
color = (0, 0, 0)

# Skapa en ny surface (en blå rektangel) på 100x100 pixlar
blue_surface = pygame.Surface((100, 100))     
blue_surface.fill((0, 0, 255))  # Fyller med blå färg (RGB)

# STEG 6: Skapa spelloopen
running = True
while running:
    # STEG 7: Hantera händelser (t.ex. stänga fönstret)
    for event in pygame.event.get():
        if event.type == pygame.QUIT:   # Om spelaren klickar på "X"-knappen
            running = False             # Avsluta loopen
    
    # STEG 8: Uppdatera fönstret
    screen.fill(color)      
    screen.blit(blue_surface, (350, 250)) # Kopierar den blå surface till position (350, 250)
    pygame.display.flip()       # Uppdatera skärmen med ändringarna

# STEG 9: Avsluta Pygame
pygame.quit()       # Stäng ner Pygame
sys.exit()          # Stäng programmet helt

Förklaring för koden markerat med grön färg:

  • rad 18: pygame.Surface((100, 100)) skapar en ny surface som är 100x100 pixlar stor. Du kan tänka på detta som en mindre "duk" som vi kan fylla med färg eller annan grafik.
  • rad 19: blue_surface.fill((0, 0, 255)) fyller den nya surface med blå färg, där (0, 0, 255) är färgkoden för blå (RGB).
  • rad 31 screen.blit(blue_surface, (350, 250)) kopierar vår blå surface till skärmens surface. Här bestämmer vi positionen på skärmen där den blå ytan ska visas, i det här fallet vid koordinaten (350, 250).

Ytterligare koncept och användningsområden ​

  • Rita former på en surface: Du kan rita former direkt på en surface med funktioner som:

    • pygame.draw.circle(),
    • pygame.draw.elipse(),
    • pygame.draw.arc(),
    • pygame.draw.line(),
    • pygame.draw.polygon()

    Exempel 2

    python
    pygame.draw.circle(screen, (255, 0, 0), (50, 50), 40)  # röd cirkel

pygame.draw.elipse(screen, (0, 255, 0), (50, 50, 130, 50))  # grön elips

pygame.draw.arc(screen, (0, 255, 255), (300, 250, 170, 150), 0, 3.14, width = 5)  # grön elips

pos = pygame.mouse.get_pos()
pygame.draw.line(screen, (255, 0, 255), (400, 300), pos ) 

pygame.draw.polygon(screen, (200, 200, 200), ((100, 100), (200, 300), (500, 100), (200, 230)))

Sammanfattning ​

Pygame surfaces är grunden för all grafik i Pygame. En surface är en yta där vi kan rita grafik, text eller bilder. Huvudskärmen är en stor surface, och vi kan skapa egna surfaces för att hantera olika delar av spelet.

Viktiga koncept:

  • Skapa surfaces med pygame.Surface().
  • Blit: Kopiera en surface till en annan (oftast till skärmens surface).
  • Uppdatera skärmen med pygame.display.flip() för att visa ändringar.

Genom att förstå och använda surfaces kan vi skapa alla visuella delar av ett spel, från karaktärer till bakgrunder, och bygga spännande spelupplevelser!
