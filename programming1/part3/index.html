Kontrollera om Python är installerat ​

För att kontrollera om du redan har python installerat på en Windows-dator, kör följande kommandoraden nedan på Windows PowerShell (Terminal) eller cmd.exe (Kommandotolken).

För att kopiera kommandoraden klicka med din muspekare i den lilla sh- eller fil-ikonen till höger:

sh
python --version
1
  • Sök i startfältet i Windows efter: cmd eller Windows PowerShell
  • Kör kommandoraden ovan, python -- version, på Windows PowerShell eller kommandotolken (cmd). Se bilden nedan:

Om du upptäcker att du INTE har Python installerat på din dator, gå till nästa avsnitt!

Verifiera installationen ​

  1. Öppna en terminal (Windows PowerShell) eller kommandotolk:

    • Windows: sök i startfältet efter "Windows PowerShell" (Terminal) eller "cmd":

  2. Kontrollera Python-versionen:

    • Skriv följande kommando på Windows PowerShell/kommandotolken och tryck på Enter:
    sh
    python --version
    1
    • Detta bör visa den installerade Python-versionen.
