Installera tilläg i VS Code ​

Vi ska installera de mest grundläggande och nödvändigt tillägen för Python-utvecling i VS code:

Python från Microsoft ​

När du installerar Python-tillägget från Microsoft i Visual Studio Code får du en omfattande uppsättning funktioner som förbättrar din Python-utvecklingsupplevelse.

Installation ​

För att installera Python-tillägget:

  1. Öppna VS Code.
  2. Gå till Extensions Marketplace: Klicka på Extensions-ikonen i sidopanelen som har en fyrkant med röd färg eller tryck Ctrl+Shift+X. Se bilden nedan.
  3. Sök efter "Python": Skriv "Python" i sökfältet med fyrkanten med orange-färg. Se bilden nedan.
  4. En lista med många alternativ visas: Klicka på Python från Microsoft. Se bilden nedan.
  5. Installera tillägget: Klicka på "Install" som har en fyrkant med grön färg. Se bilden nedan:

Klicka här för att se funktionerna som man får med tilläget "Python - Microsoft"
  • Syntaxmarkering: Färgar koden för att göra den mer läsbar.
  • Kodformatering: Automatisk formatering av din kod för att följa PEP 8-standarder (kan konfigureras för andra formateringsstandarder också).
  • Kodkomplettering: Förslag på kod medan du skriver.
  • Felkorrigering: Markerar syntax- och typfel.
  • Dokumentation: Visar dokumentation för funktioner och metoder direkt i redigeraren.
  • Inbyggd debugger: Starta och felsök din kod direkt i VS Code.
  • Brytpunkter: Sätt brytpunkter, inspektera variabler och steg igenom koden.
  • Interaktiv konsol: Utför kod i realtid under debugging.
  • Kodkörning: Kör Python-filer eller valda kodblock direkt från redigeraren.
  • Terminalintegrering: Starta en integrerad terminal där du kan köra din kod.
  • Pylint, Flake8, mypy, etc.: Stöd för olika linting-verktyg för att säkerställa kodkvalitet och följa kodstandarder.
  • Felkorrigering: Visar fel och varningar i redigeraren.
  • Inbyggd teststödd: Upptäcker och kör enhets- och integrationstester med stöd för testverktyg som unittest, pytest och nosetests.
  • Testtäckning: Visar täckningsgrad för dina tester.
  • Jupyter Notebooks: Skapa och redigera .ipynb-filer.
  • Kör celler: Kör enskilda celler eller hela notebooks direkt i VS Code.
  • Interaktiv miljö: Redigera och visualisera Jupyter Notebooks utan att lämna redigeraren.
  • Rename: Byt namn på variabler, funktioner och klasser globalt.
  • Extract method: Extrahera kodblock till nya funktioner eller metoder.
  • Sort Imports: Organisera och optimera importuttalanden.
  • Virtual Environments: Stöd för att skapa och hantera virtuella miljöer.
  • Conda: Integration med Conda-miljöer.
  • Automatisk upptäckt: Upptäcker och föreslår Python-installationer och virtuella miljöer.
  • Kodsnippets: Fördefinierade kodblock för vanliga Python-mönster och konstruktioner.
  • Anpassade snippets: Möjlighet att skapa och använda egna snippets.
  • Python Interactive Window: En interaktiv konsol som liknar Jupyter Notebooks, där du kan köra och visualisera kod.

Black Formatter ​

Black Formatter är ett tillägg utvecklat av Microsoft för Visual Studio Code (VS Code) som används för att automatiskt formatera Python-kod enligt specifika stilriktlinjer. Det är baserat på Black, en populär kodformatterare för Python.

Klicka här för att se några viktiga punkter om Black Formatter tillägget:

  1. Automatisk Formatering: Black formaterar automatiskt Python-kod enligt PEP 8-stilguiden, vilket hjälper till att upprätthålla en konsekvent kodstil i hela projektet.

  2. Enkelt Att Använda: Efter installation och aktivering av tillägget i VS Code, kan du enkelt formatera din kod genom att använda kommandot "Format Document" eller "Format Selection" från kontextmenyn eller kortkommandon.

  3. Konfigurationsmöjligheter: Även om Black har starka åsikter om kodformat, kan vissa inställningar konfigureras, som maxlängd på rad (default är 88 tecken).

  4. Integrering med VS Code: Tillägget integreras sömlöst med VS Code, vilket gör det lätt att använda för utvecklare som redan arbetar i denna populära editor.

  5. Standardisering: Genom att använda Black kan team standardisera kodformatet, vilket minskar kodgranskningstid och minimerar stilrelaterade diskussioner i pull requests.

Installation ​

För att installera Black Formatter-tillägget:

  1. Öppna VS Code.
  2. Gå till Extensions Marketplace: Klicka på Extensions-ikonen i sidopanelen som har en fyrkant med röd färg eller tryck Ctrl+Shift+X. Se bilden nedan.
  3. Sök efter "Black Formatter": Skriv "Black Formatter" i sökfältet med fyrkanten med orange-färg. Se bilden nedan.
  4. En lista med många alternativ visas: Klicka på Black Formatter från Microsoft.
  5. Installera tillägget: Klicka på "Install" som har en fyrkant med grön färg. Se bilden nedan.
