Python random-modulen ​

En av de mest användbara modulerna i Python är random, som används för att generera slumpmässiga tal och göra andra slumpmässiga operationer.

Den här genomgången är utformad för gymnasiestudenter och kommer att förklara grunderna i random-modulen, ge exempel på dess användning och visa hur man kan tillämpa den i olika projekt.

1. Introduktion till random-modulen ​

För att använda random-modulen måste du först importera den i ditt Python-program. Detta gör du med följande kommando:

python
import random
1

2. Slumpmässiga tal ​

random.randint(a, b) ​

Funktionen random.randint(a, b) genererar ett heltal mellan a och b, inklusive både a och b.

python
import random

print(random.randint(1, 10))
1
2
3

random.random() ​

Funktionen random.random() genererar ett decimaltal mellan 0.0 och 1.0 (inklusive 0.0 men exklusive 1.0).

python
import random

rand_float = random.random()
print(rand_float)

# Slumpmässigt decimaltal med två decimaler
print(f"{rand_float:.2f}")
1

random.uniform(a, b) ​

Funktionen random.uniform(a, b) genererar ett slumpmässigt decimaltal mellan a och b, inklusive både a och b.

python
import random

rand_uniform = random.uniform(1, 10)
print(rand_uniform)

# Slumpmässigt tal med två decimaler
print(f"{rand_uniform:.2f}")

3. Slumpmässigt val från en sekvens ​

random.choice(seq) ​

Funktionen random.choice(seq) returnerar ett slumpmässigt element från en icke-tom sekvens (som en lista eller sträng).

python
import random

frukter = ['äpple', 'banan', 'körsbär', 'druva', 'päron']
print(random.choice(frukter))
1
2
3
4

random.choices(population, weights=None, k=1) ​

Funktionen random.choices(population, weights=None, k=1) returnerar en lista med k antal element valda från population, med möjlighet att tilldela vikter till elementen.

python
import random

frukter = ['äpple', 'banan', 'körsbär', 'druva', 'päron']
print(random.choices(frukter, weights=[10, 1, 5, 3, 1], k=2))
1
2
3
4

Slumpmässig blandning och urval ​

random.shuffle(seq) ​

Funktionen random.shuffle(seq) blandar elementen i en sekvens på plats, vilket innebär att den ändrar den ursprungliga sekvensen.

python
import random

kortlek = ['hjärter ess', 'spader kung', 'ruter dam', 'klöver knekt']
random.shuffle(kortlek)
print(kortlek)
1
2
3
4
5

random.sample(population, k) ​

Funktionen random.sample(population, k) returnerar en lista med k unika element valda från population.

python
import random

frukter = ['äpple', 'banan', 'körsbär', 'druva', 'päron']
print(random.sample(frukter, 2))
1
2
3
4

Seed-funktion för reproducerbarhet ​

random.seed(a=None) ​

Funktionen random.seed(a=None) initierar slumptalsgeneratorn. Om du använder samma seed-värde kan du reproducera samma slumpmässiga talsekvens.

python
import random

random.seed(42)
print(random.random())
1
2
3
4

Avslut ​

random-modulen i Python är ett kraftfullt verktyg som kan användas för att generera slumpmässiga tal, välja slumpmässiga element från sekvenser, och mycket mer. Att förstå och kunna använda denna modul öppnar upp många möjligheter inom olika områden som spelutveckling, simuleringar och statistik.

Jag hoppas att denna genomgång har gett dig en god förståelse för hur random-modulen fungerar och hur du kan använda den i dina egna projekt.
